Changing the Value of an int

Once an int has been declared and a value has been assigned to it there are many ways of changing its value.

int num = 7;

Variables are reusable, so if you would like to put a brand new number in, you simply have to write another assignment statement.

num = 15;

Performing Math on an int

When you want to change the value of an int, you need to be careful. Assume that the following variable has been declared:

int year = 1985;

An operator is a symbol that is used to denote that some operation should be done with two pieces of data and yield a result. The basic math functions will work using the symbols +, - , *, and /. Assuming that you would like to increase the value of the year variable by 30, you may try the following:

year + 30;

This does add the two values together, but we have been irresponsible with the value. In the end we need the value of the year variable to change. To do that looks like the following:

year = year + 30;

At first this seems weird, but you have to think of this in two distinct steps. First the RIGHT side of the equals sign is evaluated. (1985+30 is 2015). AFTER the new value is determined it then replaces the value in year, which is a very important step. If you don't re-assign to the year variable, it will still contain the value 1985. Below is a slo-mo version of what happens.

year = year + 30;
year = 1985 + 30;
year = 2015;

Shortcuts: mathmatical assignment operators

The process of performing a simple math operation and then assigning the value back into the original variable is a very common operation. For that reason there are actually several brand new operators that both perform a math operation AND assign the result back into the original value.

+= , -= , *= , /=

Back to our original example, if there is a variable named year, with a value of 1985, and you want to add 30 to it.

year += 30;

is the same statement as

year = year + 30;

but not as laborious to write. The += operator first adds the two values together and then stores the value back into the year variable. These shortcuts are very convenient when you need to quickly increase a variable.

Consider the following code:

int year = 2001;
int age = 22;
year -= age;
System.out.println("You were born in " + year);

This snippett shows a method for determining the year someone is born when you know the current year and the age of the person. The line that is red is decreasing the year variable by whatever the value of age is (22). The age variable does not change. Only the variable on the left is affected.

Super Shortcuts: the increment and decrement operators

The mathmatical assignment operators above make it easy to quickly modify the contents of a variable, but there an even quicker one specifically when when you want to add or subtract one. They are known as the increment and decrement operators

int val = 10;
val = val + 1;
val += 1;
val++;

Following a variable with a ++ is the increment operator, which increases the value of the variable it's attached to by one. The last three lines of the code above do exactly the same thing. They increment the value of val by 1.

Example:

int num = 5;
num++;
System.out.println(num);
num++;
System.out.println(num);
num--;
System.out.println(num);

The output of the above code would be:
6
7
6

The value of num changes several times. Each time a ++ is used, the value is incremented and each time a -- is uses, it's decremented.

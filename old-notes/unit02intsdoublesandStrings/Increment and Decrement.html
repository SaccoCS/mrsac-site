Increment and Decrement

 

The Increment Operator:

Once looping comes into play, we will be frequently needing to increase a variable by exactly 1 (known as incrementing).  Consider the following line of code, which increase the num variable by 1;

int num = 7;
num = num + 1;

At the end of the code, the num variable has the value 8. 

Since this is such a common process, there is a shortcut for it.  Following any int variable with a ++ will change the value inside of the variable to be one larger. This is known as the increment operator.  Using it, the previous code may be rewritten as:

int num = 7;
num++;

This code will also reassign the num variable to have an 8.

 

The Decrement Operator:

Just as ++ will add one to a variable. Using -- will subtract one from a variable.  This is known as the decrement operator.

 

int num = 7;
num--;

By the end of this code, num will hold the value 6

Getting a double from a String

Similar to with ints, we may want a way of turning a String into a double so that we can do some math with it. This is often useful when we get some user input that is supposed to be a double.

String dubStr = "3.14159";
double dub = Double.parseDouble(dubStr);

In the example above, the dubStr variable has no mathematical value, while the dub variable is a double type, so math will work properly.

String Methods 1:

For these programs, you are going to create one class with 5 static methods.  Each takes in a String as a parameter and performs some sort of task.  All of the methods will not be returning anything, just use a println.

Since each method is static and independent of the others, I recommend that you test out the methods that you've completed before starting on the others.  Use the following code skeleton to help you start.

public class StringMethods1
{

    public static void nameMeasure(String str)
    {

    }

    public static void vertical5(String str)
    {

    }

    public static void firstAndLast(String str)
    {

    }

    public static void endChopper(String str)
    {

    }

    public static void halfAndHalf(String str)
    {

    }

}

 

Method Descriptions:

1) nameMeasure - takes in a name as a parameter (String), and prints out a sentence that indicates how long the name is

Sample input: Sacco
Output: Hello Sacco,  There are 5 letters in your name.

2) vertical5 - takes in a 5 letter String as a parameter and uses 5 printlns to print a letter on each line.  You may assume that there are exactly 5 letters in the String.

Sample input: Sacco
Output:
S
a
c
c
o

3) firstAndLast - takes in a String as a parameter and outputs the first and last letters of that String.

Sample input:  Mister Sacco
Output: M o

4) endChopper - takes in a String as a parameter and prints out the String with the first 3 letters and the last 3 letters removed.

Sample input: Mister Sacco
Output: ter Sa

5)  halfAndHalf - takes in a String as a parameter and uses two printlns to output the first half of a String on one line, and the second half on the following line.  You may assume that the number of letters in the String is even.

Sample input:  abcdefghijklmn
Output:
   abcdefg
   hijklmn

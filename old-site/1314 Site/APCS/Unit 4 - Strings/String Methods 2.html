sentenceMaker - Asks the user to enter a String, then prints it out in the style of a sentence, meaning the first letter is capitalized, and the rest of it is lower case.

Example:
Please enter a sentence:
hElLo HOW are you?
A proper sentence would be: Hello how are you?

 

wordGrabber: Prompts the user to enter a two word String, then prints out the first word on one line and the second word on the next.

Example:
Please enter a two word String:
Hello World

Hello
World

Hint:  Finding the index of the space is important for the boundaries of your substring

getInitials - Promts the user to enter their full name ( a three word String) and prints out their initials

Example:
Please enter your full name:
Ian Danger Sacco
Your initials are: I.D.S.

Hint:  The indexOf method will only tell you the location of the first space.  If you use substring to remove the first word and the space following it, you can use indexOf again to locate the following space.  For example:  "Ian Danger Sacco" has a space at index 3.  Removing the first word gives us "Danger Sacco", which will be referring to the space between 'Danger' and 'Sacco'

 

goFalcons: Prompts the user to enter a String that contains the String “srhs”. The program will then print out the entire String with only the SRHS capitalized.

Example:
Please enter a String:
fncswiouasrhsopwfelkdn
fncswiouaSRHSopwfelkdn

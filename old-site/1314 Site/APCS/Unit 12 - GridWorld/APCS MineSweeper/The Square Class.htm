Square is a Class that represents the covered square on the minesweeper grid. 

Square extends Actor, but also has two instance fields.  One to keep track of if the square contains a bomb or not, and another that will keep track of if the Square has already acted this turn ( you will set this to true every time you act, and after every time, the runner will set it back to false.)

The Square constructor will take in a boolean that will be true if the Square is a bomb and false if not.

Add these simple methods:

public boolean isABomb()      //returns true if this Square contains a bomb

public int numBombsAround()      //returns the number of surrounding actors that are either a Bomb or a Square that isABomb

public void reset()     //  sets your ‘already acted’ variable to false.

public boolean alreadyActed()                   //returns true if this actor has acted recently

public String toString()      //this is what shows when you hover on the grid.  Use it to tell you if it’s a bomb or not.

 

The act method:

The act method will be the method that is called when the Square is clicked.  The first line of the act method should set your ‘already acted’ variable to true.  If you don’t it will crash your program.

There are three situations that a square can be in, which depend on how many bombs surround the Square.  Override the act method to figure out the situation the Square is in and act accordingly.

1)  If the Square is a bomb, then it will create a new bomb and have it put itself in the grid at the Square’s Location.  This will trigger the World to end the game.

2)  If the Square is not a bomb and is surrounded by one or more bombs, then it will put down the PlainRock extension that corresponds to the number of bombs that surround it ( new One(), new Two(), etc.).

3) If the Square has no bombs surrounding it, then it will tell all of the actors around it who have not acted to act().  After that, put a new Zero() in place of the Square.

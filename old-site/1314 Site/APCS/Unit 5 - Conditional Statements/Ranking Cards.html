Ranking Cards

In many games it is important to know how the cards compare to the other cards in the game.  This is where you are going to use the getRank method that you wrote earlier ( Two = 2, Three = 3, ... Jack = 11, Queen = 12, King = 13, Ace = 14) .

Add a method to the Card class named isHigherRank, which takes in a Card as a parameter and returns true if the current object (this) has a higher rank than the parameter.

public boolean isHigherRank( Card other )

Remember, the goal is for the following code to compile and run.  One card is asked how it compares to another.

Card c1 = new Card("JD");    //Jack of Diamonds
Card c2 = new Card("TS");    //Ten of Spades

if( c1.isHigherRank( c2 ) )
{
   System.out.println( c1+" is a higher rank than "+c2);
}
else
{
   System.out.println( c1+" is not a higher rank than "+c2);
}

Add methods named isLowerRank and isEqualRank which will look very similar to the isHigherRank method.

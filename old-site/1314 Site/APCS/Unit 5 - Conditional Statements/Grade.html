Grade

Create a class named Grade, which will be used to represent a Grade on a report card.

The only instance field a Grade object should have is the String that represents the Grade.  It will contain Strings like "A-" or "C+" or even just "B"

Add a constructor that will take in a String as a parameter and assign it directly to the instance field.  Beware: the String may be 1 or 2 characters long.  Try to avoid IndexOutOfBounds exceptions.

gpaVal

Add a method named gpaVal that will take in no parameters and use the instance field to return a grade point value based on college rules.  In college, an A is worth 4 and a B is worth 3 like normal, but plusses and minuses count for .3.

For example:  A C is worth 2 points, but a C+ is worth 2.3, and a C- is worth 1.7.

There are also exceptions to the rule.  An A+ is only worth 4.0, and anything starting with F is a 0.

Hint:  Don't simply write a gigantic chain of else-ifs.  Create a local variable and use the first letter to assign 4,3,2, or 1.  After that, check the second character to see if you should add/subtract a .3

toString

The toString method should simply return the instance field.

compareTo

Once you have tested with the first runner, create a method named compareTo which takes in a Grade and returns an int according to the following rules:

if 'this' Grade's gpaVal is lower than the parameter's gpaVal, return any negative number
if 'this' Grade's gpaVal is greater than the parameter's gpaVal, return any positive number
if 'this' Grade's gpaVal is equal to the parameter's gpaVal, return 0

To test your compareTo method, use the following runner.  It will create 50 random grades and sort them from highest to lowest using your compareTo method.  If the second list is unsorted, your compareTo needs work.

import java.util.ArrayList;

public class GradeRunner2
{
    public static void main()
    {
        ArrayList<Grade> grades = new ArrayList<Grade>();
        ArrayList<Grade> unsorted= new ArrayList<Grade>();
        
        System.out.println("Creating 50 random grades...");
        
        for( int i =0; i<50; i++)
        {
            String[] grade = {"A","B","C","D","F"};
            String[] plusOrMinus = {"+","","-"};
            
            int gradeNum = (int)(Math.random()*grade.length);
            int sign = (int)(Math.random()*plusOrMinus.length);
            
            Grade tmp = new Grade(grade[gradeNum]+plusOrMinus[sign]);
            unsorted.add(tmp);
            grades.add(tmp);
            
        }
        
        
        System.out.println("Sorting using compareTo...\n");
        for(int i = 0; i < grades.size()-1;i++)
        {
            
            int bigIndex = i;
            for(int j =i; j < grades.size();j++)
            {
                
                if(grades.get(j).compareTo(grades.get(bigIndex))>0)
                   bigIndex = j;
                              
            }
            
            Grade tmp = grades.remove(bigIndex);
            grades.add(i,tmp);
            
        }
        System.out.println("Unsorted\t\tSorted");
        for(int i = 0; i < grades.size() ; i++)
        {
         System.out.println( "   "+unsorted.get(i)+" \t\t\t   "+grades.get(i));   
        }
        
    }
}

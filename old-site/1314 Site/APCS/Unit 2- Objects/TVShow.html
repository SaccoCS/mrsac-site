In-Class Practice:  TVShow

The following is an example of something you may be asked to do on the unit test.  Given the following description you should be able to determine instance fields, constructors and methods.

Write a class named TVShow, which keeps track of the title of a television show as well as the channel it’s on and the number of viewers it has. 

TVShow’s constructor should take in a String and an int, which represent the title and the channel of the TVShow.  The constructor should also set the number of viewers to 0.

Write three accessor methods:  getTitle, getViewers, and get Channel, which take in no parameters and simply return the appropriate instance field.

Write a method named changeChannel, which takes in an int from the user and assigns it to the channel variable.

Write a method named increaseViewers, which takes in an int and adds it to the number of viewers that the TVShow has.

Write a method named toString, which takes in no parameters and returns a String of the form “______ is on channel _____ and currently has _______ viewers”

 

 

Following is an example of code that would get a 100%.  The variable names and the order that the methods come in do not have to be the same, but the return types, parameter types, and general function of the methods should be the same. 

public class TVShow
{
    private String title;
    private int channel;
    private int viewers;

    public TVShow(String tmpTitle, int tmpChannel)
    {
        title = tmpTitle;
        channel = tmpChannel;
        viewers = 0;
    }

    public String getTitle()
    {
        return title;
    }

    public int getViewers()
    {
        return viewers;
    }

    public int getChannel()
    {
        return channel;
    }

    public void increaseViewers(int tmpViews)
    {
        viewers = viewers + tmpViews;
    }

    public void changeChannel(int tmpChannel)
    {
        channel = tmpChannel;
    }

    public String toString()
    {
        String str = title +" is on channel "+ channel + " and currently has "+viewers+" viewers";
        return str;
    }
}

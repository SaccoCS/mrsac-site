SuperScan

The process of using a Scanner to get an int from the user can be a little bit arduous. After constructing a Scanner it requires prompting, getting from the user, and then parsing into an int.

Scanner scan = new Scanner(System.in);
System.out.println(“Enter a number”);  //prompting
String str = scan.nextLine();  //String retrieval
int num = Integer.parseInt(str); // parsing

Since much of this process is the same each time we run it.  We are going to write a class named SuperScan, which will have methods that combine all of the prompting retrieving and parsing all in one.  Our goal in the end is to only have to do the following:

SuperScan scan = new SuperScan();
int num = scan.getUserInt("Enter a number");

Notice that the getUserInt method takes in a String and returns an int.  It takes care of all of the prompting, retrieving, and parsing for us.

You should now have two libraries that BlueJ loads.  The sacco.jar, and the My Tools project.

Creating the SuperScan Class:

Create a class named SuperScan. The only instance field a SuperScan object will have is a Scanner. It is perfectly fine to have an object as an instance field. Don’t forget to import java.util.Scanner;

private Scanner myScan;

SuperScan is deep down inside a Scanner, but the methods will be a little more useful for our purposes.  Creating a class that does this is known as wrapping the Scanner class. 

The constructor of SuperScan should construct a new Scanner and assign it to the instance field. Now that the constructor has set up the Scanner for use, the rest of the methods may use the variable without worrying about it being set up.

Methods

public String getUserString(String prompt)
This method should use System.out.println to first print out the prompt, and then return the String that the user inputs. Normally our objects don’t do the println, but this objects purpose is to do the work for us, so it’s ok.

public int getUserInt(String prompt)
This method is similar to the previous, but with one extra step. It will print out the prompt, get the String from the user, and parse it into an int before returning it.

Testing:

Once your class compiles add the following runner to your project. It does the same exact thing, but two different ways. One is the old way using a Scanner object, the other is the new way using a SuperScan object.  Read through and you'll notice that user input is a little bit easier to get now.  You may use this in any project that requires input from the user from here on out.

import java.util.Scanner;
public class SuperScanRunner
{
    public static void newWay()
    {
        SuperScan scan = new SuperScan();
        
        String str = scan.getUserString("Please enter a String");
        System.out.println("The returned String is: "+str);
        
        int num = scan.getUserInt("Please enter an int");
        System.out.println("The returned int is: "+num);
      
    }
    
    public static void oldWay()
    {
        Scanner scan = new Scanner(System.in);
        
        System.out.println("Please enter a String");
        String str = scan.nextLine();        
        System.out.println("The returned String is: "+str);
        
        System.out.println("Please enter an int");
        String intStr = scan.nextLine();   
        int num = Integer.parseInt(intStr);
        System.out.println("The returned int is: "+num);
    }
}

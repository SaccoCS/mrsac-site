Part 2:  Getting ints

Complete the following programs.  Each of them follows the same general strategy:
- Construct a Scanner
- Prompt the user to enter the number
- Use nextLine to retrieve a String version of the number
- Pass the String into Integer.parseInt to get an int version
- Perform some sort of math operation (+,-,*) and store the result in a new variable
- println out the results

1)  Write a Java program named Favorite that asks the user for their favorite number, then outputs your favorite number, which happens to be double the input number.  The asterisk (*) is used for mutliplication.

For example:
Enter your favorite number:
14
Well, my favorite number is 28

 

2)  Write a Java program named Adder that will ask the user for two numbers, then output the sum of the two numbers.

For example:
Enter the first number:
5
Enter the second number:
15
The sum of 5 and 15 is 20

 

3)  Write a program named Squarer, which will ask the user for a number, and then print out its square. 

For example:
Enter a number:
6
6 squared is 36

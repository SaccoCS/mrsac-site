Fighter

Write an Object class named Fighter to represent a character in a game. Have instance fields to keep track of the Fighter’s health points, a String for his ‘special attack’ and a String for his name.



Create a constructor that takes in two Strings and an int as parameters (in that order) to set to the instance fields. In order to match the way it was used in the runner, the name of fighter must be declared first, followed by the special move, and the health points will be third.

You will need the following methods in your Fighter class:

toString don’t include the special move in you toString, but include the name and health points

getHealth returns the health points of the fighter

takeDamage(int points) reduces the hit points by the specified amount. This method returns nothing.

getSuper returns the name of the Super move

getName returns the name of the fighter

win returns a victory phrase in the form of a String (ex. return "Hooray!";)

lose returns a losing phrase in the form of a String



Runner:

public class FighterRunner
{

   public static void main()
   {
      Fighter player1 = new Fighter("Ryu","Haduoken",100);
      Fighter player2 = new Fighter("Blanka","Electricity",100);


      System.out.println(player1+"\n"+player2+"\n");


      System.out.println(player1.getName()+" hits "+player2.getName()+" with his "+player1.getSuper()+" attack");

      player2.takeDamage(45);

      System.out.println(player1+"\n"+player2+"\n"+"\n");







      System.out.println(player2.getName()+" hits "+player1.getName()+" with his "+player2.getSuper()+" attack");

      player1.takeDamage(25);

      System.out.println(player1+"\n"+player2+"\n"+"\n");





      System.out.println(player1.getName()+" hits "+player2.getName()+" with his "+player1.getSuper()+" attack");

      player2.takeDamage(55);

      System.out.println(player1+"\n"+player2+"\n"+"\n");



      System.out.println(player2.getName()+" loses: ");

      System.out.println(player2.lose());

      System.out.println(player1.getName()+" wins: ");

      System.out.println(player1.win());



   }
}

      System.out.println("What is "+name+"'s special move?");

      name = scan.nextLine();
      System.out.println("What is "+name+"'s special move?");
      special = scan.nextLine();

      fighters[1] = new Fighter(name, special, 100);

      //determine who goes first
      int num = (int)(Math.random()*2);

      }

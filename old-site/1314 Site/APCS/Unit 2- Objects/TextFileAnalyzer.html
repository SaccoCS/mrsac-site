TextFileAnalyzer

TextFileAnalyzer API

The TextFileAnalyzer in sacco.jar is an object that loads a text and turns it into a String.  It can then return the word count or tell you how many times a single character String appear in it.  The word count is determined only by using spaces, so it is not 100% accurate.

Warning:
1)  The text file must be a .txt . When you supply the name of the file, include the .txt extension.
2)  The file name is case sensetive, so be careful.
2)  The text file must be in the exact same folder as the .java file that you're working with.  Not any subfolders.

If you read the constructor, you will notice that a TextFileAnalyzer's constructor requires that the file name be passed in as a parameter.  It will attempt to load the file on construction, but crash if it cannot find it.

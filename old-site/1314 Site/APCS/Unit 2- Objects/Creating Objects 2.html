Creating Objects 2

Create the following classes in your Creating Object project

1)  Date

Create a class named Date that represents a particular day.  Create three instance fields.  You'll need a ints for the month, day and year.

The Constructor will take in THREE parameters.  This means that the constructor's header will look like public Date( int tmpMonth, int tmpDay, int tmpYear)

What this new constructor header means:
Previously we've constructed objects in the same way:
ShapeOrganizer shapeOrg = new ShapeOrganizer();
Store myStore = new Store();
Car myCar = new Car();

Date is different.  Since the constructor requires parameters, you will not be able to say Date myDate = new Date();  You MUST provide three ints:
For example:  Date myDate = new Date(2,22,1983);
This means that Dates have no default values, they are simply constructed with the values provided at creation.

Write a toString method that returns a String in the form "m/d/y", e.g. 1/1/1900

Test out all your methods, using a toString to make sure they are functioning.

2) Point

Create a class named Point, whose job is to keep track of a two dimensional point.  Create instance fields to represent the x coordinate and the y coordinate of the Point

The Constructor should take in two Parameters, an x and a y, which should be passed into the appropriate instance fields

Add the following methods:

setX, setY:  These two methods set the value of the instance field to whatever is passed in.

getX, getY: These methods will take in no parameters and will return the appropriate instance field

Add a method named translate which takes in two ints as parameters and adds the first parameter to the x instance field and the second parameter to the y instance field.

Create a toString method  (takes no parameters and returns a String) which returns a String in the form "(2,3)"  where the 2 is the x coordinate and the 3 is the y coordinate

- Create a Point object that contains the point (5,8)

- use the getX and getY methods to get the individual data out, and then print out a string in the form "The x coordinate is now _____, and the y is _____".  Though this is essentially the same purpose as the toString, do not use the toString.

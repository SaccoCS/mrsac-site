Square

Create a class named Square, which will be in charge of holding all of the locations and dimensions of a square. Create instance fields to represent the x and y, as well as the side length and the color (use a String for the color)

Create a constructor that takes in no parameters and sets the values of x and y to 0 as well as the side length to 100 and the color to green.

Create a SECOND constructor that takes in three values, an x, a y, and a side length. This constructor should also set the default color to blue.

Add the following methods to the Square class:
- methods setX, setY, and setSideLength which will take in an int from the user and assign it the appropriate instance field.
- method setColor, which takes in a String from the user and assigns it to the color instance field
- method getArea, which takes no parameters and returns the area of the square.
- method getPerimeter, which takes in no parameters and returns the perimeter of the square
- method doubleSide, which takes in no parameter and doubles the side length

- a toString method which returns a String in the form “This _color_square is located at (__,__) and a side length of ____. Its perimeter is ______ and its area is _______
Note: Don’t recalculate the perimeter and the area in this method. You can actually call the other methods of Square from within the class. You use the special word this instead of a variable
Example:
int per = this.getPerimeter();

Generally you don’t want to repeat code that performs a task that a method already performs.



Write a runner class that does the following:
- Construct a Square with using the default constructor (... = new Square()).
- print out the toString
- double the side length
- print out the toString
- Construct a SECOND Square with a different variable, this time use the constructor using the values of 25,40, and 80 (... = new Square(25, 40, 80) )
- print out the toString
- set the color to Red
- print out the toString



The output should look something like this:

This green square is located at (0,0) and has a side length of 100. Its perimeter is 400 and its area is 10000
This green square is located at (0,0) and has a side length of 200. Its perimeter is 800 and its area is 40000
This blue square is located at (25,40) and has a side length of 80. Its perimeter is 320 and its area is 6400
This red square is located at (25,40) and has a side length of 80. Its perimeter is 320 and its area is 6400

Using Methods that Return Values

TodoList API

Copy and paste the following program into a new project.  Read through each line and observe how each method is used.  Pay attention to how the returned values are handled.

import sacco.*;
public class TodoListRunner
{
    public static void main()
    {
        //Constructs a new TodoList into the myList variable
        TodoList myList = new TodoList();

        //Adds three items to the list
        myList.addItem("Milk");
        myList.addItem("Eggs");
        myList.addItem("Chicken");
        myList.addItem("Bread");
        myList.addItem("Cereal");

        //Calls the toString method and prints the returned value
        String list = myList.toString();
        System.out.println(list);

        //Gets the item count and prints it out
        int count = myList.getItemCount();
        System.out.println("There are "+count+" items in the list\n");

        //Removes item number 2, then prints out the toString
        myList.removeItemByNumber(2);
        list = myList.toString();
        System.out.println(list);

        //Gets the item count, then prints it out
        count = myList.getItemCount();
        System.out.println("There are "+count+" items in the list\n");

        //Marks item number 2 and 4 as complete, and then prints out the toString
        myList.completeItemByNumber(2);
        myList.completeItemByNumber(4);
        list = myList.toString();
        System.out.println(list);

        //Removes all completed items from the list and prints out the toString
        myList.removeCompletedItems();
        list = myList.toString();
        System.out.println(list);
    }
}

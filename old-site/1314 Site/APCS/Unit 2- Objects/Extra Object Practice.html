Extra Object Practice

Create a class named Distance, which will be responsible for holding a number and a unit (such as 5 inches).  Create two instance fields, an int to represent the value, and a String to represent the units.

The Distance constructor should take in two parameters, an int and a String, and apply them to the proper instance fields.

Write two accessor methods, getValue and getUnits, which will simply return the appropriate instance field.

Write two mutator metohds, setValue and setUnits, which will take in a parameter and assign it to the appropriate instance field.

Write a method named increaseValue, which will take in a parameter and increase the instance field by that much.

Write a toString method that will return a String in the form of "The distance is _____ _____(s)".  For example "The distance is 4 mile(s)"

 

Solution:

public class Distance
{
    private int value;
    private String units;
    
    public Distance(int tmpVal, String tmpUnits)
    {
        value = tmpVal;
        units = tmpUnits;
    }
    
    public void setValue(int tmpVal)
    {
        value = tmpVal;
    }
    
    public void setUnits(String tmpUnits)
    {
        units = tmpUnits;
    }
    
    public int getValue()
    {
        return value;
    }
    
    public String getUnits()
    {
        return units;
    }
    
    public void increaseValue(int tmpVal)
    {
        value = value + tmpVal;
    }
    
    public String toString()
    {
        String str = value+" "+units;
        return str;
    }

}

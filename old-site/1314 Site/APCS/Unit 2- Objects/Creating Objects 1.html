Creating Objects 1

1)  Store

Create a Java class named Store.  The Store class will have two instance fields, a String to hold the name of the store and a String to hold the item that is on sale.  The Constructor will also set the default sale item to “Eggs” and the default name to "Vons".  Write a method named toString that takes in no parameters, and returns a String in the form of "Welcome to ______, today's sale item is ______"

Add a method named setSaleItem, which takes in a String as a parameter and assigns it to the saleItem instance field. 

 

Once the Store clas compiles, make a runner that will do the following:

- Construct a new Store Object
- Call the toString method and print out the result
- Call the setSale item and change it to "Milk"
- Call the toString method and print out the result

 

2) Car

Create a class named Car.  The Car class will have two instance fields, a String to hold the license plate of the Car, and an int to hold the mileage.  The Constructor should set the default license plate to "DEALER" and set the mileage to 0.

Add a method named setLicensePlate which takes in a String as a parameter and assigns it to the license plate instance field.

Add a method named drive which takes in an int as a parameter and adds it to the current mileage.

Note:  Adding to a variable can feel strange.  If you have an int x that you want to add five to, it would look like this:

x = x + 5;
The right side is calculated first, and then the result is assigned back into x.

If you have an int x and you want to add an int num to it, it's similar:

x = x + num;

Add a toString method that will return a String in the form "The car with license plate ______ has ______ miles on it"

 

Once the Car class compiles, create a runner that will do the following:
- Construct a new Car object
- set the license plate to "CS4LIFE"
- print out the toString
- call the drive method and pass in 50
- print out the toString
- call the drive method and pass in 99
- print out the toString

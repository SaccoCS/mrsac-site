Changing Bases

One way to convert to binary is to use mod and int division repeatedly:

1)  Mod your number by 2.  The result is the next largest digit.

2)  Divide your number by 2 using int division.  The result is your new number.

3)  Repeat steps 1 and 2 until your number is 0.

 

For Example:  11

11 % 2 = 1

11%2 = 1 Binary number so far:  1
11/2 = 5 New number:  5
Repeat  
5%2 = 1 Binary number so far: 11
5/2 = 2 New number: 2
Repeat  
2%2 = 0 Binary number so far: 011
2/2 = 1 New number: 1
Repeat  
1%2 = 1 Binary number so far:  1011
1/2 = 0 New number: 0

 

Create a class named ChangingBaseMethods, which will have the following public static methods:

public static String getBinary(int dec)

This method's job is to take in a Decimal number and use a while loop to go through the process above to return a String that represent the binary version of the number.

Strategy:
- Before you loop, create a String variable and assign to it an empty String ( a String with no characters in it ).
- as long as dec is greater than 0, use mod get the next binary digit and add it to the front of the String. After that, divide dec so that it's half the value it used to be.
- after the loop has finished, return the String

Converting back:

We have a method that will help us go from decimal to any base, the only thing left to do is to write a method that will go in the other direction.

Converting numbers to decimal is all about the place value:

For Example:  A2 in hexadecimal has two digits, each worth a different place value. 

the A is in the 16's place: Since A is worth 10, 10*16 =>160
the 2 is in the 1's place: 2*1 => 2

160+2 = 162

 

 Add the following method:

public static int baseToDec(String num, int base)

The first parameter is a number, but since it can be any base it must be taken in as a String ("1B4A9" cannot be an int).  This method will convert it to its decimal value and return it as an int.

STRATEGY:
In order to get the decimal version of the number, you must get the digits one at a time, multiply them times their place value, and add them to the instance field. I recommend starting an int for the place value and starting it at the largest place value needed.  It's based on the length of the number and base, you should use Math.pow(and cast it to an int) . 

Then use a while loop that will take the front digit, multiply it by the place value, add it to the instance field, and then divide the place value by the base, making it smaller for the next digit. Once the place value is 0, your loop is complete. Make sure that the instance field for the base also gets set.

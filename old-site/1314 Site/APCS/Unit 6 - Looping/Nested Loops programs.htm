Nested Loops programs

 

For the following programs, any inputs should be taken in using parameters.  Any output will be prints and printlns.

Reminder:  Text that's printed out using System.out.println will appear on the next line.  Text that's printed using System.out.print will be on the same line.

    		How many rows: 6

		1 2 3 4 5 6
		1 2 3 4 5 6
		1 2 3 4 5 6
		1 2 3 4 5 6
		1 2 3 4 5 6

    		How many rows: 6

		1

		4 3 2 1
		5 4 3 2 1
		6 5 4 3 2 1

  3. Write a program named SquareDesign that will generate a Square Design like the one below for any input values for N (N is the number of rows and columns of Xs in each square). For example: 

    		N:  5

		I-X-X-X-X-X-I

		I-X-X-X-X-X-I
		I-X-X-X-X-X-I
		I-X-X-X-X-X-I
		+-----------+

    Hint:  The top and bottom will require a non-nested for loop to print out the appropriate number of hyphens.  The magic number is 2*n+1.

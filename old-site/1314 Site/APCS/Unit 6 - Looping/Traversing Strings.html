Traversing Strings

Create one class named TraversingStrings and add the following static methods to it.  All input will be handled through parameters.  Download the TraversingStringsTester.class file and move it to your project folder.  When you restart BlueJ you should see a TraversingStringsTester class with the words "no source" on it.  This class has testers for all of the methods provided, plus one method that will test all of them.  This is required for turning in the programs.

 

isACap - This method will take in a single character String and return true if that character is a Capital Letter.  This method is a helper method.  It doesn't involve traversing, but may be called from other methods to save time on programming.

 

isAllInts - This method will take in a String and return true if the String contains only numbers and no letters.

 

containsACapital - This method will take in a String and return true if any of the characters are a capital letter

 

isAllSameCharacter - This method will take in a String and return true if every character in the String is the same.

 

largestInteger - This method will take in a String that is made up entirely of integers and return the largest one (as an int)

 

indexOfLargestInteger - This method will take in a String that is made up entirely of integers and return the index of the largest one (you may need more than one local variable for this one)

 

getGPA - This method will take in a String that is made up of a student's grades and return a double that represents the GPA of the student.  For Example:  the String "AABACA" would return a 3.5

 

justTheCaps - This method will take in a String and return a new String that is made up only of the capital letters from the parameter String.
For Example:  "AbcDeFg" would return a "ADF"

 

catCount - Mr. Sacco has lost his cat in the parameter of this method.  Take in a String, and return the number of times that the word "cat" appears in the String.
For Example:  "catcaaacattacat"  would return a 3.

 

countPeaks - A peak is defined as a capital letter surrounded by a lower case letter on each side.  This method takes in a String and returns the number of peaks that it contains
For Example:  "AsdFgHJkLmN"  would return a 2.

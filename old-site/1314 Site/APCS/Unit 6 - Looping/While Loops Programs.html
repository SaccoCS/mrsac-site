While Loops Programs

Create one class named WhileLoops, which will contain the static void methods below.  Each method will have no parameters, so all input should be taken in using Scanner.

Methods:

oddCounter - Prompt the user to enter a number, then print out all ODD numbers that are less than that number.

Example:
Enter a number:
10
The odd numbers less than 10 are:
1
3
5
7
9

Useful Method Alert:

Normally we use scan.nextLine() to get a String from the user:
String str = scan.nextLine();

If we want an int, we need one more step.  The Integer.parseInt(_____) method will take whatever String is passed as a parameter and attempt to convert it to an int.  If the String cannot be made into an int, the program will crash.

String str = scan.nextLine();
int num = Integer.parseInt(str);   //now the int version of whatever was passed in is stored in the num variable

There is also a method for Doubles
String str = scan.nextLine();
double num = Double.parseDouble(str);   //now the double version of whatever was passed in is stored in the num variable













verticalString - Prompt the user to enter a String, then print the String out one letter at a time until there are no more letters.  The length of the String should not matter.

Example:
Enter a String:
Hello
H
e
l
l
o

sentenceSplitter - Prompt the user to enter a sentence.  Output the sentence one word at a time until there are no words left.  This should work for sentences of ANY length.  Hint:  Use indexOf(" ") to drive your while loop.  Don't forget what it returns when the String cannot be found.

Example:
Enter a sentence:
Mr. Sacco is totally awesome
Mr.
Sacco
is
totally
awesome

safeDivider - Prompt the user to enter a number, then a number to divide by.  Generally, the program should output the quotient and remainder of the division.

Example:
Enter a numerator:
20
Enter a denominator:
3
20 divided by 3 has a quotient of 6 with a remainder of 2

Make a loop for the program that will protect the program from dividing by zero.  If the user enters a zero in the denominator, print out "Please enter a non-zero denominator" and ask again.

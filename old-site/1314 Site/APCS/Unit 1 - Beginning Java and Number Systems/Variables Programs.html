Variables Programs

Complete the following programs that output Strings and ints.  Each program should use at least one variable.

1)  Write a program named FavNum, which stores your favorite number as an int, and then uses a println to print it out.  DO NOT JUST System.out.println("My favorite number is 22");  Create an int variable to hold the number, and then concatenate it.

For example:
My favorite number is 22.

 

2) Write a program named FavColor that displays your name and favorite color on separate lines. Place your name on the first line and your favorite color on the second line below.

For example:
My name is Willie.
My favorite color is Blue.

 

3)  Write a program named Summer, which stores three different ints into three different variables.  First print out what the three values are, and then print out the sum of all three (calculate the sum in the code).

For example:
My three numbers are 2,8, and 4
They sum to 14

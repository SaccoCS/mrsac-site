Scribbler

Write a program that lets the user draw on the panel using the mouse. To accomplish this, you must have instance fields that represent the last x and y of the mouse. To draw lines you must use the onMouseMove to draw a line from the LAST x and y (instance fields) to the current ones (m.getX() and m.getY()).   The last thing that the onMouseMove method should do is put m.getX() and m.getY() into the x and y instance fields.  This is so when the method is called again, the lastx and lasty have the previous values.

This should only happen while the mouse is being held down. Each time the mouse is dragged, a new line is drawn to the background Picture.
 
Like before, you will also need to use the onMousePress and onMouseRelease to determine when the mouse is pressed. When it is, you should store the x and y in your instance fields. This is so when you start dragging, the first line is drawn from the correct spot, not 0,0.
 
 
Determining left vs right click:
So far we've only used the getX and getY methods of the MouseEvent class to help determine where the event occurred. There is also another method called getButton(). If you call the getButton method it will return an int that represents which of the mouse buttons was pressed. 1 is the left button, 2 is the middle button (pressing the scroll wheel), and 3 is the right button.
 
Modify your onMousePress method so that a right click will 'erase' the screen. This can be accomplished by drawing a giant white rectangle on the background Picture.

Creating Custom SCanvas

import sacco.*;
import sacco.gui.*;

PaintBrush API

If you run the following main method, you won't be very impressed:

public static void main()
{
   SFrame frame = new SFrame();       
   SCanvas pan = new SCanvas();
   frame.add(pan);
   frame.setSize(500,500);
   frame.setVisible(true);
}

 

Just like the main method says, there's a frame with a blank panel on it... BOOORRIIINNG.

Using a technique called inheritance, we can make our own customized panels that look however we want.

 

1)  Create a new class with the following header:

public class DemoCanvas extends SCanvas

Notice the extends keyword.  This means that we want a DemoCanvas object to act just like an SCanvas.  If we want it to act any differently, we define it here.  We actually want to change the way that this SCanvas draws, or paints, itself.  This means we must Override the method that does it.

2)  Add a method with the following header:

@Override
public void paintCanvas(PaintBrush brush)

Since we're replacing the method that the SCanvas already has, you have to be extremely careful about your spelling.  Any different and it won't work.  The idea here is that every time the canvas paints itself, this method is called and a PaintBrush object is passed in.  Our job is to use the PaintBrush to draw what we want it to look like.  PaintBrush has many different methods that have to deal with painting.  Copy the following code into the paintCanvas method to see some examples of what it happening:

PaintBrush API

brush.setColor( Color.RED );
brush.fillRectangle( 50,150, 70, 35);    //x,y,width,height
 
//Creates a Blue Rectangle outline
brush.setColor( Color.BLUE );
brush.drawRectangle( 150,10, 10, 35);     //x,y,width,height
 
//Creates a Green Circle
brush.setColor( Color.GREEN );
brush.fillOval( 200, 300,100,100);  //x,y,width,height
 
//Draws some Black Lines
brush.setColor(Color.BLACK);
brush.drawLine(275,20,400,100);     //x1,y1,x2,y2 connects (x1,y1) to (x2,y2)
   
//Prints out a Purple "Hello World!!!
brush.setColor(Color.MAGENTA);

brush.drawString("Hello World!!!",250,200);

 

3)  Adding the custom canvas to the frame:  So far we have a canvas which does not do anything on its own.  Just like with SCanvas, they must go on a frame.  Use the following runner method:

public static void main()
{
    SFrame frame = new SFrame();       
    DemoCanvas pan = new DemoCanvas();
    frame.add(pan);
    frame.setSize(500,500);
    frame.setVisible(true);
}

Notice that the canvas being added is a DemoCanvas, which is the special canvas that we just created.  Run it and see what happened:

Each of those PaintBrush commands changed how the canvas looks.  We can now draw simple pictures on our frames.

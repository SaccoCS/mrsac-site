Polygon Objects

You can draw/fill polygons by creating a new Polygon object and adding all of the points of the vertices of the polygon you want to draw using the method addPoint(x, y). Once all the points have been added, you may use brush.fillPolygon or brush.drawPolygon. The points may be emptied from the Polygon using the clear() method.
 
The order of the points is important. They should be listed in ORDER, either clockwise or counterclockwise. If you don’t, the points will criss-cross and it will look terrible.
 
Check out the following demo that adds points to a Polygon which the paintCanvas method then paints on the screen.

import sacco.*;
import sacco.gui.*;
 
public class PolygonDemoCanvas extends SCanvas
{
 
    private Polygon myPoly;
 
    public PolygonDemoCanvas()
    {   
        myPoly = new Polygon();  
 
        myPoly.addPoint(200,50);
        myPoly.addPoint(350,150);
        myPoly.addPoint(350,350);
        myPoly.addPoint(50,350);
        myPoly.addPoint(50,150);
 
        
    }
 
    public void paintCanvas(PaintBrush brush)
    {
        brush.setColor(Color.WHITE);
        brush.fillRectangle(0,0,500,500);
 
        brush.setColor(Color.RED);
        brush.fillPolygon(myPoly);
 
    }
    
    public static void main()
    {
        SFrame frame = new SFrame();
        frame.setLayout(new BorderLayout());
        frame.add(BorderLayout.CENTER, new PolygonDemoCanvas());
        frame.setSize(500,500);
        frame.setVisible(true);
    }
 
}

Assignment:

PolygonCreator: Create a program that will allow users to create polygons on the screen. This program should have instance fields for the Polygon's Color, the background image, and the "current" Polygon.
 
The paintCanvas method will first paint the background image, then paint the "current" polygon to the screen. If a person left clicks, all you have to do is add that point to your current polygon. If a person RIGHT clicks, you have to paint the current polygon to the background, then reset the current polygon ( use Polygon's reset() method )and change the polygon color.

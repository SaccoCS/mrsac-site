Polygon Creator

 

Create a program that will allow users to create polygons on the screen.  This program should have instance fields for the Polygon's Color, the background image, and the "current" Polygon.

 

The paintCanvas method will first paint the background image, then paint the "current" polygon to the screen.  If a person left clicks, all you have to do is add that point to your current polygon.  If a person RIGHT clicks, you have to paint the current polygon to the background, then reset the current polygon ( use Polygon's reset() method )and change the polygon color.

Polygon Objects

You can draw/fill polygons by creating a new Polygon object and adding all of the points of the vertices of the polygon you want to draw using the method addPoint(x, y).  Once all the points have been added, you may use brush.fillPolygon( ) or brush.drawPolygon( ).  The points may be emptied from the Polygon using the clear() method.

 

The order of the points is important.  They should be listed in ORDER, either clockwise or counterclockwise.  If you don’t, the points will criss-cross and it will look terrible.

 

Check out the following demo that adds points to a Polygon which the paintComponent method then paints on the screen.

 

import javax.swing.*;

import java.awt.*;

 

public class PolygonDemoCanvas extends SCanvas

{

 

    Polygon myPoly;

 

    public PolygonDemoPanel()

    {  

        myPoly = new Polygon(); 

 

        myPoly.addPoint(200,50);

        myPoly.addPoint(350,150);

        myPoly.addPoint(350,350);

        myPoly.addPoint(50,350);

        myPoly.addPoint(50,150);

 

       

    }

 

    public void paintCanvas(PaintBrush brush)

    {

        brush.setColor(Color.WHITE);

        brush.fillRect(0,0,500,500);

 

        brush.setColor(Color.RED);

        brush.fillPolygon(myPoly);

 

    }

 

}

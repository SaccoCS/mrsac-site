import java.util.ArrayList;

ArrayList Notes:

Here's an example program that uses many of ArrayLists methods, put it into BlueJ and read through the comments.

ArrayLists are objects that can hold multiple things. To set up an ArrayList, you declare it like so:

ArrayList strList = new ArrayList();

The add method:

Now the variable strList is set up to hold many Strings. At the moment the list has nothing in it. We can add things to it using the add method.

strList.add("Hello");

Now str has one String in it at index 0. Each time the add method is used, the String is added at a new index, so if I make two more commands:

strList.add("World");

strList.add("!!!");

"World" is added at index 1, and "!!!" is added at index 2, resulting in the table below.

index 0 1 2 value "Hello" "World" "!!!"

The get method:

You can use the "get" method to retrieve the String at a given slot.

String tmp = strList.get(1);

by saying get(1), we are asking the strList for the String at index 1. After this line is excecuted the String "Hello" is stored in tmp.

Traversing through the ArrayList

Suppose I wanted to print out every String in strList.

for( int i =0; i< strList.size() ; i++)   //loop through every index in the String
{
   String tmp = strList.get( i );         //get the i'th String out of the list
 
   System.out.println( tmp );          //prints out that String
}

The remove method:

You can use the remove method to remove a String from the list entirely.

if strList is below index 0 1 2 value "Hello" "World" "!!!"

strList.remove(1); removes the element in slot 1 ("World") from the list. All the elements that come after index 1 are shifted to fill the gap that's created.

index 0 1 value "Hello" "!!!"

The enhanced add method:

We can add strings at a specific index if we use the other version of the add method that takes in an int before the String.

strList.add(1, "Sacco"); adds the String "Sacco" to index 1. Everything at index 1 and above is shifted to one index higher.

index 0 1 2 value "Hello" "Sacco" "!!!"

ArrayLists can hold any type of Object, but requires a small twist for ints and doubles. An ArrayList of ints shoud be declared of type Integer and doubles need to be capitalized:

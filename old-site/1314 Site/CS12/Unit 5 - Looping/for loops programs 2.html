Helpful Code:

 

1) Write a Java program named averager that prompts the user for 10 numbers, and outputs their average. Do not repeat your Scanner commands 10 times. Write a for loop that will loop 10 times.

Hint:  You need a double that will be used to sum all of the user's numbers.  Don't forget that division doesn't work the same way when you're dividing ints.  You will need to involve a double in the calculation.

 

 

2)Write a Java program named Factors that prints out all the factors of a number. For example if the number is 30, the output will look like:

 The factors of 30 are: 1, 2, 3, 5, 6, 10, 15, 30.
Hint:  You can use mod to determine if a number is a factor of the original number.  Use a System.out.print instead of System.out.println in order to print only on one line.  Make sure that you don't output an extra comma at the end.



3)Write a program named PerfectNumber. A perfect number is a positive integer such that the sum of the Factors (not including the original number itself) equals the original number. Thus, 28 = 1 + 2 + 4 + 7 + 14 is a perfect number. Write a Java program that determines if a positive integer is a perfect number. Your output should be either:

The factors of 28 sum to 28
28 is a perfect number

or
The factors of 12 sum to 16
12 is not a perfect number

Hint:  First use a loop to find the sum of all the factors, and later check to see if the sum is equal to the original number.

 

4) Write a program named isPrime that inputs an int from the user, then outputs two things: The number of factors that number has (including 1 and itself) and then it will tell if the number is prime or not.

Example:

Enter a number:
6

6 has 4 factors.
6 is not prime

or

Enter a number:
11

11 has 2 factors.
11 is prime

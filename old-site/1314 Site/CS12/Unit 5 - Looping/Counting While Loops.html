While Loops that Count

Each of the programs below will use a while loop, but also require at least one more variable that doesn't control the loop; it only keeps track of some information that should be read back after the loop has finished.  Each time declare the int you want before the loop starts and change it as the loop runs.

1)  OddCounter:  Write a program named OddCounter that will repeatedly ask the user to enter numbers.  This will only stop once the user has entered a -888.   This is known as a flag value because it signals to the program that it can stop looping.  After the loop, the program should print out the number of odd values that were entered.

For example:
Please enter a number:
4
Please enter a number:
1
Please enter a number:
5
Please enter a number:
6
Please enter a number:
10
Please enter a number:
9
Please enter a number:
-888

You have entered 3 odd numbers

 

2)  MaryCounter: Write a program that asks the user for a first name repeatedly until the user enters the flag value "XXX". Output how many people whose name is "Mary" that were entered by the user.

Hint:  To check to see if a variable str does NOT have "XXX", use the exclamation point in front of the .equals method.  !str.equalsIgnoreCase("XXX")

For example:
Please enter a name:
Steve
Please enter a name:
Mary
Please enter a name:
Anne
Please enter a name:
Sue
Please enter a name:
Mary
Please enter a name:
Helen
Please enter a name:
XXX

Mary was entered 2 times

3)  GradeStats:  Write a program named GradeStats that repeatedly prompts the user to enter the percent that a student got on an exam.  This program should use the standard grade scale to determine the grade that each student got and keep track of all the A's, B's, C's, D's, and F's that were earned.  After a flag value of -1 has been entered, print out how many of each grade was recieved.

For Example:
Please enter a percent:
75
Please enter a percent:
99
Please enter a percent:
82
Please enter a percent:
71
Please enter a percent:
33
Please enter a percent:
79
Please enter a percent:
-1

Grade Statistics:
A's: 1
B's: 1
C's: 3
D's: 0
F's: 1

4)  BioPopulation:  In a biology experiment, Mr. Sacco finds that a certain type of bacteria doubles its population every 12 hours.  Write a program that will ask the user for the starting population, and then calculate how many hours it will take for that population to reach 1,000,000 or more.  Hint:  You will need a variable to hold the population, but also one to count how many times the loop has to run until the population is past 1,000,000. 

For Example:
Enter a starting population:
1000

It would take 120 hours to pass 1,000,000

5)  NumSum:  Write a program that will ask the user to input a number, and then calculate the sum of all the numbers from 1 up to and including that number.

Please enter a number:
10
The sum of the numbers from 1 through 10 is 55

Hint:  You will need three ints.  One to hold the user's input, a counter that increases by 1 each time, and a sum variable that the count gets added to each time the loop runs.

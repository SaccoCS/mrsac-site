Finding Factors:

 

To find factors of an int, you have to remember the 'mod' operation %.
mod returns the remainder of the two numbers ex: 7%2 --> 1

If a number is a factor of another, the mod will evaluate to 0
ex. 12%4 --> 0

This can be used in an if statement to help determine if a number is a factor of another.

ex: num1 and num2 are ints

if( num1 % num2 == 0 )

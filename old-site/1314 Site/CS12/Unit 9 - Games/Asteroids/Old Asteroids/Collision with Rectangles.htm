Collision using the Rectangle Class

 

 

 

In order to see if things are colliding, we are going to get some help from the Rectangle class, which has a method named intersects that returns true if one Rectangle is intersecting another.  This is considered a ‘hit box’ for those of you that are gamers.

 

 

We are going to need to modify our Asteroid and Bullet class to have a method that returns a Rectangle with the same x, y, width, height as the Object.

 

public Rectangle getRectangle()

{

    Rectangle tmpRect = new Rectangle(x,y,width, height );

    return tmpRect;

 

}

 

The actionPerformed of the Panel is going to need to check every Bullet against every Asteroid.  This involves for loops within for loops.

 

for( int i = bulletList.size()-1; i>=0; i--)

{

      Bullet tmpBul = bulletList.get(i);

      Rectangle bulRec = tmpBul.getRectangle();

 

      for( int j = asteroidList.size()-1; j>=0; j--)

      {

           Asteroid tmpAst = asteroidList.get( j );

           Rectangle astRec = tmpAst.getRectangle();

 

            if( bulRec.intersects( astRec ) )

            {

                 bulletList.remove( i );

                 asteroidList.remove( j );

                 break;

            }

      }

 

}

The Spaceship

 

Create a new class named Spaceship.  The spaceship will have many of the same instance fields and methods as Asteroid, but with a couple more.

 

One important distinction is that the Spaceship class’ rotation is the same as its direction, so you should choose one variable to represent both of them.

 

Instance Fields:

x

y

speed

direction

width

height

image

 

Methods:

constructor

int getX()

int getY()

int setX(int tmpX)

int setY(int tmpY)

int getWidth()

int getHeight()

void update()

void paintMe(Graphics g)

 

NEW METHODS:

 

void turnLeft() and void turnRight():  changes the direction variable by about 20

void speedUP():  adds a small amount to the speed ( about 3 )

void slowDown():  subtracts an even smaller amount from speed ( about .75 or 1).  Add an if statement to make sure the speed doesn’t drop below zero.

 

 

To add the Spaceship to the panel, create a single instance field variable for your ship.

 

private Spaceship myShip;

 

Then you need to add code in all the same places

Constructor

paintComponent

actionPerformed

 

 

ADDING THE KEYLISTENER:

 

The KeyListener is similar to the smooth paddles of pong.  Use booleans to represent if the keys are held down or not.  Then use the timer to either say myShip.speedUp(), myShip.slowDown(), myShip.turnLeft(), or myShip.turnRight().

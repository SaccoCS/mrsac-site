Collision with the Ship:

Similar to the Asteroid and Bullet class, add a getHitBox method to the Spaceship class.  This will allow us to know when the ship collides with an asteroid.

In the onTimerTick method of the Canvas, add a section that checks to see if the ship collides with any of the Asteroids in the list.  It is similar to the loop that checks the Asteroids against the Bullets, but you only need one loop instead of two.  You will first need to get myShip.getHitBox(), and then loop through all of the Asteroids to see if they overlap.  If they do, don't remove anything.  Simply stop the Timer ( myTimer.stop()).  If you run this program, it should completely stop the moment that the ship hits an asteroid.

Add a void method named reset which does each of the following:
- Constructs a new ArrayList into the asteroidList instance field.  This will clear any Asteroids from the screen.
- Adds 4 Asteroids to the asteroidList
- Starts the timer again (myTimer.start())

Modify the onKeyPress method so that when the R button is pressed, the reset method is called.  Also modify the Constructor so that it starts the game with only 4 Asteroids.

 

Adding a level system:

The last bit of making this a little more game-like is to add a level system.  The goal is to have each level start out with increasingly more Asteroids until it overwhelms the ship. You'll need a few new bits of code to help you out.

- Add an int instance field named level.  Make sure that it starts at 1.  Also go into your reset method and have it set the level back to 1.

- The level is over when there are no more Asteroids left.  This means that asteroidList.size() == 0.  In the Canvas' onTimerTick method, check to see if the size is zero.  If it is,  make the level variable go up and then write a loop that will add more asteroids (similar, but not the same, as the loop in the constructor).  The number of asteroids to add should be 2*level+3.  This will cause two more asteroids for each level increased.

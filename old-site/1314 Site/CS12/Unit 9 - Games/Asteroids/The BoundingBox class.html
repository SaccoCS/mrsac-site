The BoundingBox class

Now that we have all of the important objects on our Asteroids Canvas it's time to think about collision.  The most important thing that we're going to need is the ability to tell when two rectangles overlap.  In this assignment you are going to create an object that will represent a rectangle and be able to tell if it overlaps with another rectangle.  This object will be different than the SpaceShip or Asteroids class because it is not meant to be a visual object.  This means that the canvas will NOT have an ArrayList<BoundingBox>.  It will be used differently.

 

Create a class named BoundingBox, which will be responsible for representing a rectangle that surrounds an object.

Add doubles for the x,y,width, and height of the box.

The constructor should take in four ints and apply them to the appropriate instance fields

Add the basic accessor methods: getX(), getY(), getWidth(), and getHeight()

 

The overlaps method:

The most important thing that a BoundingBox is going to be used for is to tell if it overlaps another BoundingBox object.  Add the following method header.

public boolean overlaps(BoundingBox other)

 

This is a terrifying method.  It is in the BoundingBox class, but the parameter of the method is also a BoundingBox.  You must think of it as two separate BoundingBoxes.

1)  The BoundingBox that we are writing.  You can access it's information using the word 'this'.  For example, this.getWidth() and this.getX() should return the width and the height of "this" BoundingBox

2) The parameter BoundingBox.  This is a completely separate BoundingBox object.  It has its own x,y,width, and height.  It has a variable named  other, so you can use that to acces the information.  i.e. other.getY() and other.getHeight().

 

The idea is to use the x,y,width and height of both 'this' box and the 'other' box to see if they overlap adn return true if they do.  This can be tricky to do, unless you look at it a specific way.

Instead of thinking "What is true when two boxes overlap?", you should think "What is true when they are NOT overlapping?".  If two boxes are NOT overlapping, at least one of four things must be true:
1)  The other box is completely to the right of this box (other's left edge is greater than this' right edge)
2)  The other box is completely to the left of this box (other's right edge is less than this' left edge)
3)  The other box is completely above this box (other's bottom edge is less than this' top edge)
4)  The other box is completely below this box (other's top edge is greater than this' bottom edge)

BE CAREFUL:  It is never just comparing an x and another x.  In order to be completely to the right, an x must be greater than an x+width

Your method should look for any of these cases and return false.  If none of those return, then the boxes must be overlapping.

 

Copy and paste the following runner, which has two BoundingBox objects.  The center box turns red when your overlaps method returns true.  Move the mouse until the two boxes overlap.  When they do, the center box should turn red.  Move the mouse all around to make sure that it only turns red when the boxes are overlapping.

   

import sacco.*;

public class BoundingBoxTester extends SaccoWindow
{
    private int x,y;
    private BoundingBox stationary, drag;
    
    public BoundingBoxTester()
    {
        x = Integer.MAX_VALUE;
        y = Integer.MAX_VALUE;  
        stationary = new BoundingBox(170,200,160,100);    
    }

    public void paintWindow(Canvas brush)
    {

        if(drag != null && drag.overlaps(stationary))
        {
            brush.setColor("RED");
            brush.fillRectangle(170,200,160,100);    
        }
        
        brush.setThickness(4);
        brush.setColor("black");
        brush.drawRectangle(170,200,160,100);    

        
        brush.drawRectangle(x-50,y-80,100,160);
    }

    public void onMouseMove(MouseEvent m)
    {
        x=m.getX();
        y=m.getY();
        drag = new BoundingBox(x-50,y-80,100,160);
    }

    public static void main()
    {
        BoundingBoxTester b = new BoundingBoxTester();
        b.showWindow();
    }
}

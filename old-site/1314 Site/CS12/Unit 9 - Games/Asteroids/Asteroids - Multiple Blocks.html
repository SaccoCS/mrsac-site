Asteroids: Multiple Blocks

Now that you have one Asteroid running, modify your program to use ArrayLists instead of just a single variable. This process is the same as with multiball, so if you're stuck you should look back at the Converting to Multiball assignment.


Make sure that in the constructor of the Asteroid class you set random values to your x,y,speed,direction, and size ( width and height should be the same if you want squares ).

x and y - The random values for x and y should be chosen based on the width and height of the canvas

direction and rotation should be random out of 360

width and height should be the same value. We will deal with the specifics of the random values later, but for now just randomize between 30 and 50.


When you think you're ready, double-check to make sure that you've randomized properly. Make sure that the blocks are moving in every direction (all 360 degrees), they don't all start at the same spot, and they're all different sizes. Mr. Sacco will be checking to see if you've done this when you turn it in.

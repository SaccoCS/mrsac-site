Moving the Paddles

 

In order to make the paddles move, we need to set our program up to listen to the keys.  The key listening methods are already built in; we just need to add them to the canvas:

 

@Override
public void onKeyPress(int keyCode)
{

}
@Override
public void onKeyRelease(int keyCode)
{

}

 

 

A common mistake is to try to move the paddle every time that a specific key is pressed.  Trying this will result in a clunky control system where you have to rapidly press a key in order for it to move.

 

THE KEYLISTENER METHODS WON’T MOVE THE PADDLES DIRECTLY.  DO NOT PUT ANY CODE IN EITHER OF THE KEY METHODS THAT CHANGES THE COORDINATES OF THE PADDLES

 

Instead we are going to use a combination of the Key Listener methdos and the Timer that you’ve already set up.

 

1)  Set up booleans as instance fields to represent the current state of each key you want to use ( I used up, down, w and s ).  Our goal is to have these booleans be true during the time that their key is being held and false when it’s not.  This will be practically the same process we used to see if the mouse button was being held in previous assignments


2)  We are going to modify the onKeyPress and the onKeyRelease methods. 

 

The onKeyPress method should check the  keycode to see if the key is one of our boolean instance fields.  If so, the boolean should be set to true.

 

The onKeyRelease method should do the same thing, except set the booleans back to false.

 

What is this doing?  Our booleans start out set to false.  When a user presses a button, the keyListener will set its boolean to true, indicating that it is being held down.  This will remain true as long as that key is still being held down, because the moment that the key is released, the boolean is set back to false.

 

3)  The onTimerTick is what actually does the moving.  After all of the moving of the ball and the paddle checking is complete, there should be an if statement for each boolean that checks to see if it’s true.  If so, the paddle moves in the direction that the key represents.

 

Result:  As long as the key is held, the paddles will move at a consistent and smooth rate.

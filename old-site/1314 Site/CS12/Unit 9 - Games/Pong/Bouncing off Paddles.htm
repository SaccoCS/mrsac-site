Bouncing off Paddles

 

Add a couple rectangles that are the right size to represent the paddles on the screen.  Use an instance field to represent the y coordinate, so that later it will be easier to move the paddle.

 

For our simple pong game, the paddles will cause the ball to do a horizontal bounce whenever the ball intersects the paddle.

 

This complicates the if statement because instead of x being just less than one side of the paddle, it also has to be greater than the other side.  The same is true for y, making your if statement need 4 separate parts.

 

 

Don't forget to account for the width of the ball AND the paddle when creating your if statements.

 

Set up two stationary paddles and test to make sure they bounce when they hit the paddles and continue on to the wall when they don't.  After that, give the ball a yVector and see how it moves.  There is a problem that will occur in very specific situations.  Read about THE VIBRATION PROBLEM and follow the steps to solve it

Testing out the variables:  It is important that you use variables to represent the y coordinate of each paddle.  Make sure that you use those variables both when you draw the paddles as well as in each of the two if statements that check for bouncing off of the paddles.  To test this out, first get the ball bouncing off of the paddles, and then change the instance fields so that the paddles are in a completely different location and try again.  Again, make sure the ball bounces off of the new location of the paddles.

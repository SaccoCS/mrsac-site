Determining the State - Moving

In the PlatformerCanvas, the booleans for left and right can be useful for telling which way we're facing and if we're moving or standing. We're definitely moving if either the left or the right booleans are true, but not both. If both are being held, we are standing.

Determining the State - Facing

To tell which way our character is facing, we need to know if left or right was the last key pressed. This is simple enough. Add a boolean to the PlatformCanvas class name facingRight and set it to true. In the PlatformCanvas onKeyPress method, go to the if statement where the right arrow was pressed. In this case facingRight should be set to true. When the left arrow is pressed, facingRight should be set to false. This is slightly different than the left/right booleans that we have now.

The Ball Class

By now you've set up a Canvas to use the Ball class to store information. Now it's time to create the Ball class. Read through what makes an Object class and then create a class named Ball in your project.

Instance fields:
Each Ball needs an x, y, xVector, and yVector. If you want, you may set up a Color variable.

Create a constructor that takes in two ints:
The two ints should be assigned to x and y, then some random vectors should be put into the xVector and yVector. This is where you would set the Color variable.

Write these other methods to either change instance fields or return information:

int getX() returns the x value

int getY() returns the y value

void update() adds the vectors to x and y

void setX(int tmpX) sets x to the given tmpX

void setY(int tmpY) sets x to the given tmpY

void horizBounce() multiplies the xVector by -1

void vertBounce() multiplies the yVector by -1

void paintSelf(PaintBrush brush) uses brush to paint this Ball at x and y
 
Once the Ball class compiles it can be used in any of the programs in the same project.

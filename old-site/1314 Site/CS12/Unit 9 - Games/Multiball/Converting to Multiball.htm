Converting to MultiBall
import java.util.ArrayList;

At this point your program has been converted to no longer use x,y,xVect, or yVect from your Canvas class.  All of the data has been created with the Ball objects we’ve created.

The reason that we’ve done this is because a Ball object is much easier to put into an ArrayList.  This is how we’re going to create multiple Ball objects on the same screen.  We need to go back and re-edit the program to use these lists.  First let’s remember about ArrayLists.

Reminders about ArrayLists:

1)  ArrayLIsts hold many objects under the same variable name

2)  Objects can be added to the ArrayList using the add method and removed with remove( int index)

3)  You can use the get(int index) method to retrieve an Object from the ArrayList without removing it from the list

4)  For loops are great at doing the same thing to each part of the list.

 

A special for loop for processing the entire list:

Assume you have an ArrayList<String> named aList.  You want to loop through the ArrayList<String> and print out the uppercase version of every String. 

Old way:

 

for( int i = 0; i < aList.size(); i++)

{

   String str = aList.get(i);

   System.out.println(str.toUppercase());

}

New way:

 

for( String str : aList )

{

   System.out.println(str.toUpperCase());

}

 

for( String str : aList ) reads as: For Each String inside of aList, temporarily name it str and run the code, then change str to the next String in the list.

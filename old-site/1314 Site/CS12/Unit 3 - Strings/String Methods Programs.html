String Methods Programs:

Each of the following programs will use one of String's methods to get the appropriate information and then print it out.

String's methods:

.length()  - returns an int that represents how many characters there are in the String variable

String str = "Hello";
int len = str.length();
After this code runs, len contains the value 5

.indexOf(String) - returns an int that represents the first appearance of the given String.  Don't forget that the first index is index 0.  If the given String cannot be found, this method returns a -1.

String str = "Hello";
int ind = str.indexOf("o");
After this code runs, the variable ind contains the value 4

.substring(int, int)  - returns a new String that represents a portion of the current String.  The first parameter is the STARTING index, the second is ONE PAST THE LAST INDEX.

String str = "Hello World";
String part = str.substring(1,4);
After this code runs, the variable part contains the value "ell".  Notice that index 4 was not included

 .toUpperCase() - returns a new String that represents the current String, but in ALL CAPS
.toLowerCase() - returns a new String that represents the current String, but in all lowercase.

String str = "Hello World";
String upper = str.toUpperCase();
String lower = str.toLowerCase();
After this code runs, the variable upper contains "HELLO WORLD" and lower contains "hello world"

1.  NumLetters - Write a program named NumLetters that will ask the user to input their name, and then output the number of letters in their name.  Do not simply output the number.  Make it a part of a sentence.

Example:

Please enter your name
Bill
Bill, there are 4 letters in your name

 

2. Shouter - Write a program named Shouter that will ask the user to enter a String, and then output the same String with all of the letters capitalized and with at least 3 exclamation points on the end.

Example:

Please enter a String
Hello World
If you were screaming it would say HELLO WORLD!!!

 

3. Sports - Write a program named Sports that will ask the user to input their favorite sports team, and then output the first letter on line one, the second letter on line two, and the third letter on line three.

Example:

Enter your favorite sports team:  Padres
The First letter in Padres is:  P
The Second letter in Padres is:  a
The Third letter in Padres is: d

4. Artist - Write a program named Artist that will ask the user to input their favorite artist, and then output the first and last letters of the artists name on the same line.  Strategy:  Use the length method to help you determine what the index of the last letter is.  The last index is one less than the length.

Example:

Who is your favorite artist?
Picasso
The first and last letters of Picasso are: Po

5. FullName - Write a program named FulName that will prompt the user three times to enter their first, middle, and last name.  The program should then output the user's initials

Example:

What is your first name?
John
What is your middle name?
Wilkes
What is your last name?
Boothe
Your initials are J.W.B.

 

6. Middle - Write a program named Middle that will ask the user to input any word, and will output its middle letters without the first and last letter.  Strategy:  First use the length method to help figure out what the last index is, and then take a substring starting at the second index and going up to, but not including the last one.

Example:

Please enter a word:
football
The middle letters are ootbal

Hint:  The substring that will give the middle letters will always begin at the same location.  The ending location is trickier, because we do not know how long the input string will be.  Review the lastLetter method in the StringHolder class and observe how the length() method is used to find the index of the last letter.

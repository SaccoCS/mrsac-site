String Methods 2

Create a separate program for each of the following.  Use a Scanner for your inputs, and don't forget the import statement.

import java.util.Scanner;

1)  FindTheX - Write a program named FindTheX, which will ask the user to input a String, and then print out the index of the first x that it finds.

Example:

Please enter a String:
saxophone
The x is located at index 2

2)  SentenceMaker - Write a program named SentenceMaker, which will ask the user to input a String, and then print it out like a sentence with the first letter capitalized and the rest in lower case.

Example:

Please Enter a String:
HelLo WoRLD
In sentence form that would say: Hello world

Strategy:  The key is to separate the first letter from the rest of the String.  Create a String named firstLetter and use a substring to put the first letter into it.  Create a String named theRest and put everything but the first letter in it.  Once you have those, you can use the toUpperCase and toLowerCase methods to finish.

3)  FirstWord - Write a program named FirstWord, which will take in a String of many words from the user and then print out only the first word.

Example:

Please enter a String:
Computer Science Rules
The first word in that sentence is Computer

Strategy:  The important piece of information here is the location of the first space in the String.  Use the indexOf method and pass in a String with just a space in it " ", store the result of than in an int named spaceIndex.  Now you can use spaceIndex to get a substring from the beginning to the first space.

4)  CapitalA - Write a program named CapitalA, which will take in a String from the user and print it out with the first "a" capitalized

Example:

Please enter a String:
Scripps Ranch High School
Scripps RAnch High School

Strategy:  The input String needs to be separated into two parts:  The part before the first a, and the part after the first a.  In order to create these we will need to figure out what the location of the first a is.  Create an int named aIndex and use the indexOf method to get the index of the a.  Once you have that, create two Strings named before and after.  Use aIndex in a substring to get the part of the String that was before the a and another to get the part that was after the a.  At the very end you can put them back together with before + "A" + after

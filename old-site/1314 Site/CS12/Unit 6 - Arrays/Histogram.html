Histogram:

The programs below relate to traversing through a given array.  Similar to traversing through a String, set up your looping variable to represent each index of an array.

For example:  if I have an array arr, and I want to create a new array where each value is double the original array

public static int[] doubleArr(int[] arr)
{
   int[] newArr = new int[arr.length];   //creates a new array the same length as arr
   for( int i =0; i < arr.length; i++)
   {
      int val = arr[i];     //gets the value out of arr at index i
      newArr[i] = 2*val;    //doubles the value and stores it in the new array at the same location.
   }

   return newArr;
}

By the end of this loop, newArr has been filled with values that are twice as big as the ones in arr.

public static int[] normalizeArray(int[] arr, int maxValue)
This method will create a new array of the same size as arr, but all the values are proportional to a maxValue.

For example: if the array is {1,2,3,4,5} and the maxValue is 50, then the returned array will be {10,20,30,40,50}, but if the maxValue was 25, then the values would be {5,10,15,20,25}

Strategy: The very first thing you should do is use the previous methods to figure out the largest value in the array. You should create a new array to hold the new values and then start looping through arr. One at a time you should get the value from arr, multiply it by maxValue and then divide by the largest value that you found earlier (it's important that you divide last because of how int division works). This new value should go into the new array at the same index. Don't forget to return the new array after the loop has finished.

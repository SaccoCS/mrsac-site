2) Create a program named BigSmallAve that takes in 5 ints from the user and stores them into an ArrayList. After the 5 ints are entered the program will output the largest and smallest values, then the average of the data. When programming, focus on one of the outputs at a time, trying them all at once can be overwhelming.


For Example: if the numbers were 1,4,6,3,5
The output would be:
The smallest value is: 1
The largest value is: 6
The average is 3.8

 

How to determine what the largest value in an ArrayList is.

a)  Before looping, create an int called largest and put the first value of the ArrayList into it.  At this point, largest will probably not contain the largest value.

b)  While looping, use the get method to grab the int at each index.  If this value is larger than the value currently in largest, then that value should be put into largest.

c)  Since the value of largest is only replaced when a larger value is found, at some point the largest value of the ArrayList will be stored there.  After the largest value of the ArrayList is stored into largest it will never get replaced.  Therefore, after the loop has run completely largest will contain the correct value.

 

3)  Create a class named CardLists and add the following method:

public static ArrayList<String> generateCardList()

This method's job is to create and return an ArrayList<String>  made up of all 52 cards in a deck.  DO NOT DO THIS USING 52 add statements.

Strategy:
You will need the following arrays to make this method easier.  

String[] vals = {"Two","Three","Four","Five","Six","Seven","Eight","Nine","Ten","Jack","Queen","King","Ace"};
String[] suits = {"Clubs","Diamonds","Hearts", "Spades"};

Each time you add a new String, it should be made up of one String from vals, and one String from suits.  Since you are traversing through two arrays you will need nested loops to loop through each one.  This time, set up the outer loop's variable to represent all the indices of the suits array, and the inner loop's variable to represent all the indices of the vals array.  Each time the loop runs, get a String from each array and add it to the ArrayList named cards.

 

Testing:  ArrayLists are harder to see in BlueJ.  To test if your method works, add the following method which will take the list from the generateCardList method and print it out.  You should see all 52 cards printed out.

public static void printCardList()
{
   ArrayList< String > cardList = generateCardList();
   System.out.println(cardList);
}

 
4) Add a method to the CardLists class with the following header:

public static ArrayList<String> shuffledCards()

This method will start by calling the generateCardList method to get an ArrayList<String> that represent all of the cards. After that is set up create a new empty ArrayList of Strings. One at a time remove a random String from the first ArrayList and add it to the new one. Eventually the new ArrayList will have all of the Strings from the first array in a shuffled order.  Return that ArrayList.



REMINDER: if I want a random number from 0 to 5, it would look like this

int rand = (int)(Math.random()*6);

Do not use 6 when calculating the random number.  You should use the current size of the ArrayList.  That way it will always return a number from 0 to size-1, exactly what is needed.

Warning:  If your for loop says   for(int i =0; i<cardList.size(); i++), this will only process half the Strings in cardList.  This is because as you remove cards, cardList.size() shrinks.  By the time i is halfway through the list, the size is half of what it originally was and the loop will quit.

One solution is to store the size of the ArrayList into a variable before you start looping and use that variable as the boundary of the loop.

int listSize = cardList.size();
for(int i =0; i<listSize; i++)

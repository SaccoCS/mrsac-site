Else if Programs

Use a else-if chains to complete the following programs:

 

1)  SmallEven:  Write a program that will ask the user for an int, and then print out the most appropriate one of the following:
"Even"
"Even and under ten"
"Neither"

When testing this program make sure that a number like 6 only prints out "Even and under ten" and not both.

Test with the following and make sure you get the appropriate output:

13  -->  Neither
8 --> Even and under 10
14 --> Even

 

2)  GoodMorning:  Write a program named GoodMorning, which will prompt the user to enter what time it is in military format ( 1300 is 1 pm) and respond with either "Good Morning", "Good Afternoon", or "Goodnight" based on the time entered.

Morning is from 400 to 1200
Afternoon is 1201 to 1800
Night is 1800 to 2359 and 0 to 399

Hint:  Night is tricky because it's split, but if you put it as the else portion of an else if chain, it will run any time it's not Morning or Evening.

Please enter a time:
700
Good Morning

Please enter a time:
1645
Good Afternoon

Please enter a time:
834
Good Morning

 

3) TestGrade:  Write a program named TestGrade, which will ask the user what percent they got on a test, and then print out the letter grade.

90 --> 100 is an A
80 --> 90 is a B
70 --> 80 is a C
60 --> 70 is a D
0 --> 59 is an F

 

4)  TaxBracket:  People owe a different percentage in taxes depending on how much money they made.  Write a program that asks the user how much they made last year and outputs how much they owe in taxes based on their income.  Don't worry about formatting your doubles.

0 --> 60000   Owe 25%
60000 --> 100000 Owe 30%
Everyone else owes 35%

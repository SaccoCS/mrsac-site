Character Analysis with booleans

When using if statements, a true/false expression is put in.

if( x > 3)
{
   System.out.println("Greater than 3!");
}

The x>3 is a boolean value, which is a data type that is either true or false.  Just like with ints and doubles, you can create variables to hold boolean values.  Those variables can then be used in an if statement.

int x = 7;
boolean isGreater = x>3;        // isGreater recieves a value of true
if( isGreater )
{
   System.out.println("Greater than 3!");
}

 

You can also get boolean values from some methods.  String's .equals method returns a boolean.

boolean sameString = str.equals("Hello");

We've used helper methods before as a tool to help accomplish a task.  In the following programs it will be useful to know what the ASCII value of a character is.  Copy and paste the following method.  You don't need to know the specifics of what's going on; just know that if you pass in a single-character string, it will return it's ascii value.  Notice that the header says "int" instead of "String".  That's because this method's job is to return an int.

public static int getASCII(String character)
{
int ascii = (int)character.charAt(0);
return ascii;
}

for example, the following will end with a 97 in the asciiVal variable

String str = "a";
int asciiVal = getASCII(str);

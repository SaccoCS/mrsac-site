Random Numbers Programs

Use random numbers to help you create the following programs. Don't forget:

int r = (int)(Math.random()*howMany)+startingValue;

1) DiceRoller: This program should simulate rolling two six-sided dice. It should print out the value of each die and also print the total.
For example:
You rolled a 1 and a 5, which gives you a 6.

You rolled a 5 and a 6, which gives you an 11.


2) GuessingGame: This number will choose a random number from 1 to 5 and as the user to guess it. It should print out its number and tell the user if they guessed it right or not.

For Example:
Please guess a number from 1 to 5:
3
Incorrect, it was 4

Please guess a number from 1 to 5:
2
Correct! It was 2


3) RandomCard: This program will choose a random playing card. Use one random number to decide the value (Ace, Two, Three, Four, Five, Six, Seven, Eight, Nine, Ten, Jack, Queen, King) and one random number to decide the suit (Clubs, Diamonds, Hearts, Spades). You do not need to do any printing inside of your if statements. You should make Strings for the suit and value, and then use the random values to assign Strings into those variables. After that is done you can just print out the value and the suit.

For example:
I have chosen a Jack of Diamonds
I have chosen the Ace of Spades


4) Magic8Ball: This program will simulate shaking a Magic 8-ball, which gives you a random answer to a question. The original 8 ball had 20 responses, but you can limit your program to 10: 4 Affirmative, 3 Neutral, and 3 Negative.

Affirmative:
- It is certain
- It is decidedly so
- Without a doubt
- Yes – definitely
- You may rely on it
- As I see it, yes
- Most likely
- Outlook good
- Yes
- Signs point to yes

Neutral:
- Reply hazy, try again
- Ask again later
- Better not tell you now
- Cannot predict now
- Concentrate and ask again

Negative:
- Don't count on it
- My reply is no
- My sources say no
- Outlook not so good
- Very doubtful


5) RandomLetter: This program will ask the user to enter a String, and then return a random letter from that String. Hint: The length of the String will help you choose an index, and substring will help you get the letter at that index.
Please enter a String:
Hello World
I have chosen 'e' as my favorite character

Please enter a String:
Mr. Sacco
I have chosen 'S' as my favorite character

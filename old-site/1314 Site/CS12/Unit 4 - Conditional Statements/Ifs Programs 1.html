Ifs Programs 1

1)  PinNumber:  Write a program named PinNumber, which will prompt the user to enter an int.  If it matches 1337, print out "Access Granted"  otherwise print out "Access Denied"

2)  EvenOdd:  This program should prompt the user to enter a number, and then tell them if it's even or odd.  Hint:  If you mod a number by 2 and the result is a 0 the number is even.  If the result is a 1 the number is odd.

3) ValidInput:  Write a program that prompts the user to enter a positive number.  If a positive number is input, print out "That number is a valid number", otherwise print out "Invalid Input:  Shutting Down"

4)  ValidEvenOdd:  This program is a combination of EvenOdd and ValidInput.  Prompt the user to enter a positive number.  If the number is positive, tell the user if it's even or odd.  If the number is not positive, print out "Invalid Input: Shutting Down"

In order to complete this, you will need to put and if statement inside another if statement!  I've given you the structure, but you have to fill in the code

if( /*number is positive*/ )
{
   
   if( /*number is even*/ )
   {
      //number is positive and even
   }
   else
   {
      //number is positive and not even
   }
}
else
{
   //number is not positive

}

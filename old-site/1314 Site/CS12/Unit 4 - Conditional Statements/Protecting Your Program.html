Protecting Your Program

There are many ways to accidentally crash your program. The following programs have you perform a task that has a chance of crashing if the wrong input is entered. Use if statements to make sure that the program will run without crashing.
 

SafeDivider - This program will prompt the user to enter ints for the numerator and denominator of a fraction, and then calculate the quotient and the remainder.  The dangerous case is when the denominator is zero.  Since dividing by zero is impossible, use an if statement to print out "undefined" when this occurs.

For example
Please enter a numerator:
35
Please enter a denominator:
4
35 / 4 = 8 r. 3

Please enter a numerator:
35
Please enter a denominator:
0
35 / 0 = undefined

 

SafeSlope - Write a program which asks the user for the coordinates of two points on a graph and calculates the slope of a line passing through them. DANGER:  Remember that the slope of a line can be undefined when the denominator of the slope formula is zero.  If the slope is undefined, do not calculate it.  Instead use the word "undefined"

 
 
            Sample input data and results:
            Enter first coordinate's x value:  2
            Enter first coordinate's y value:  5
            Enter second coordinate's x value:  4
            Enter second coordinate's y value:  6
 
            The slope passing through (2,5) and (4,6) is 0.5
 


            Enter first coordinate's x value:  3
            Enter first coordinate's y value:  7
            Enter second coordinate's x value:  3
            Enter second coordinate's y value:  -9
 
            The slope passing through (3,7) and (3,-9) is undefined
 
 
 
SafeQuad -Previously you've written a program that solves the Quadratic formula.  The problem was that only certain values of  a,b, and c have real solutions that can be calculated.  This is when the part under the square root ( b2-4ac ) is positive.  Modify the program so that it will work essentially the same when real solutions are possible, but if not, it simply prints out "There are no real solutions"

For Example:

Sample input data and results:
Enter the value for the a coefficent: 1
Enter the value for the b coefficent: -2
Enter the value for the c coefficent: 15
There are no real solutions

Sample input data and results:
Enter the value for the a coefficent: 1
Enter the value for the b coefficent: -12
Enter the value for the c coefficent: -28
The two solutions are 14 and -2

 

 SafeSummer - Create a program that prompts the user to enter a five digit number, and then caluculates the sum of all the digits.  To do this you will have to separate the String into it's individual characters before you parse.

Make sure that you protect your program from the situation where the number entered is exactly the right length.  If this is not the case, simply print out "Invalid Input: Shutting Down.

Please enter a five digit number:
12345
The sum of all five digits is 15

Please enter a five digit number:
1234
Invalid Input: Shutting Down

Please enter a five digit number:
123456
Invalid Input: Shutting Down

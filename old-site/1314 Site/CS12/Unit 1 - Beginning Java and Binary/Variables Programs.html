Variables Programs

For each of these programs, use the appropriate variable to store the necessary information, then use println’s to display it.

1) Write a Java program named FavNum that outputs your favorite number. Use only an int to display the number. Label your
results.

For example:
My favorite number is 28

2) Write a Java program name FavCol that displays your name and favorite color on separate lines. Place your name on the first line and your favorite color on the second line below.

For example:
My name is Willie
My favorite color is green

3) Write a Java program named QuickSum that will output the sum of three ints. The output is only the sum, do not display the original three numbers. Use only one System.out.println statement and label your output.

For example:
The sum is 20

4) Write a Java program named age that will output the following. The age must be output as an integer primitive type. Use only one System.out.println statement.

For example:
I am 15 years old.

5) Write a Java program named ThreeNums that creates three integer variables, assigning them each a value, then output your three integer variables as shown below. Only the commas and the words "Sum is" should be enclosed in Java string quotes.

For example:
3, 4, 5
Sum is 12

6) Write a Java program named GPA that creates a string variable to hold your name, and double variable to store your G.P.A. Have your program output the following (using your data of course!):

My name is Bob.
My G.P.A. is 3.85.

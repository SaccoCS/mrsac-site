Getting Ints From the User

For each of these programs, use the appropriate variable to store the necessary information, then use println's to display it.  Read the notes online if you do not know how to get an int from the user. 

 

1)  Write a Java program named Favorite that asks the user for their favorite number, then outputs your favorite number, which happens to be double the input number.  The asterisk (*) is used for mutliplication.

For example:
Enter your favorite number:
14
Well, my favorite number is 28

 

2)  Write a Java program named Adder that will ask the user for two numbers, then output the sum of the two numbers.

For example:
Enter the first number:
5
Enter the second number:
15
The sum of 5 and 15 is 20

 

3) Write a Java program named Ager that will take in the user's age and output how old they will be in 15 years.

For example:
Enter your age:
16
In 15 years, you will be 31 years old 

 

4)  Write a program named Squarer, which will ask the user for a number, and then print out its square. 

For example:
Enter a number:
6
6 squared is 36

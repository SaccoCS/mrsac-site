Converting to Binary from Decimal

One way to convert to binary is to use mod and int division repeatedly:

1)  Mod your number by 2.  The result is the next largest digit.

2)  Divide your number by 2 using int division.  The result is your new number.

3)  Repeat steps 1 and 2 until your number is 0.

 

For Example:  11

11 % 2 = 1

11%2 = 1 Binary number so far:  1
11/2 = 5 New number:  5
Repeat  
5%2 = 1 Binary number so far: 11
5/2 = 2 New number: 2
Repeat  
2%2 = 0 Binary number so far: 011
2/2 = 1 New number: 1
Repeat  
1%2 = 1 Binary number so far:  1011
1/2 = 0 New number: 0

 

Create a program named DecToBin, which prompts the user for a number less than 64, and then converts it to a binary String.  Use the following strategy:

Store the input number in a variable named num.

Mod num by 2 and store the result in a variable named onesPlace

Divide num by 2 and store the result back into num ( num = num / 2 )

Mod num by 2 and store the result in a variable named twosPlace

Divide num by 2 and store the result back into num ( num = num / 2 )

Repeat this process until you've put values into the onesPlace, twosPlace, foursPlace, eightsPlace, sixteensPlace, and thirtyTwosPlace variables.

Now print out the number, starting with the thirtyTwosPlace variable.  Be Warned:  If you don't include a String in your output it will literally add together all the ones and zeros.  Meaning 101101 will print out a 4.

 

For example:

Please enter a number less than 64:
41
41 in binary is 101001

Getting ints and doubles from the User

We already know that a Scanner is useful when we want to get a String from the user. We need to do a few things

  • Construct a Scanner object
  • Prompt the user for the data
  • Call the nextLine method to retrieve the data.
Scanner scan = new Scanner(System.in);
System.out.println("Please enter _____");
String result = scan.nextLine();

The problem is that the nextLine method only returns a String. When you want to get ints or doubles, there is an extra step.

Getting an int from the User

When you want to get something from the user that is not a String you will first get the data as a String and then convert it to the value that you want.

Scanner scan = new Scanner(System.in);
System.out.println("Please enter an int");
String result = scan.nextLine();

Once you have the info as a String, convert it using the Integer.parseInt method, which takes in a String and returns the int version of that String.

Scanner scan = new Scanner(System.in);
System.out.println("Please enter an int");
String result = scan.nextLine();
int num  = Integer.parseInt(result);

Whatever the result variable contains will be turned into an int and stored into the num variable, which you can then do math with.

Getting a double from the User

Getting a double is very similar to getting an int. You must first get the String version of a number, and then use the Double.parseDouble method.

Scanner scan = new Scanner(System.in);
System.out.println("Please enter an int");
String result = scan.nextLine();
double num  = Double.parseDouble(result);

After that, you can use that value as a normal double.

Using Scanner

Scanner is an object that is an alternative to JOptionPane.  It allows you to communicate with the user purely through the terminal.

Copy and paste the class below into a BlueJ project

import java.util.Scanner;
public class ScannerDemo
{
    public static void main()
    {
        //Construct a new Scanner Object
        Scanner scan = new Scanner(System.in);
        
        //Prompt the user to enter his name
        System.out.println("Enter your name ");
        
        //Call the nextLine method to grab the next String the user types
        //Store that String in a local String variable named result
        String result = scan.nextLine();
        
        //Print out a greeting, using the name that was just entered
        System.out.println("Hello "+result+", how are you?");
        
    }
    
        
}

There are three important parts:

1) importing java.util.Scanner;

2) Constructing the Scanner object. Don't forget the System.in.  It represents the terminal window.

3) Calling that object's nextLine method when you need to get a String from the user.  The program will pause an wait for you to type something and hit enter in the terminal.

 

ALWAYS PROMPT!

Since we are working only with the terminal, it is important that you print out some sort of prompt before you call the nextLine method.  Otherwise the terminal won't appear and the computer will seem to be stuck.  It won't let you compile or run anything else.  In this case you must manually restart BlueJ's Virtual Machine.

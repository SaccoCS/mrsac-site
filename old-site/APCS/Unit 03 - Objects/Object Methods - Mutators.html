Objects with Methods: Mutator Methods

Mutating the Data:

Accessor methods like getFirst and getLast mere made because reading from n.first and n.last is no longer allowed.  Another problem will occur when trying to change (or mutate) the instance fields. 

Name n = new Name("Steve","Urkle");
n.last = "Harvey";

This snippet of code is attempting to change the last name to be "Harvey" instead of "Urkle".  Similar to before, this is no longer allowed.

 

Mutator Methods:

Just like we needed to add methods to help us read the information back out of an object, we are going to have to add methods that will help us change the data.

Let's think of the method that we would like:

Name n = new Name("Steve","Urkle");
n.setLast("Harvey");
System.out.println(n.getLast());

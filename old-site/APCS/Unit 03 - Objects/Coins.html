Coins

Write a class named Coins, which will be in charge of .keeping track of a handful of coins.  You will need four int instance fields to represent the number of quarters, number of dimes, number of nickels, and number of pennies.

Add a constructor that takes in four ints and assigns them to the instance fields (quarters, dimes, nickels, and pennies).

Add four basic accessors: getNumQuarters, getNumDimes, getNumNickels, and getNumPennies

Add four mutator methods: addQuarters, addDimes, addNickels, and addPennies.  Each will take in an int and add it to the appropriate instance field.

Add the accessor method getTotal, which will return a double representing the value of all the coins together.  DO NOT CREATE A NEW INSTANCE FIELD TO HOLD THE TOTAL.  YOU DON'T NEED IT.

Add an accessor method cashFormat, which will take in no parameters and return the value of the Coins as a String using our cash formatting rules (a dollar sign and two places after the decimal).  In order to accomplish this, start by calling the getTotal method and work from there. 

Add a toString method that returns a String in the form:

"Quarters: ___
Dimes: ___
Nickels: ___
Pennies: ___
Value: ____"

Where the value is retrieved by calling the cashFormat method.

 

Copy and run the following runner: 

public class CoinRunner
{
    public static void main()
    {
        Coins purse = new Coins(1,2,3,4);        
        System.out.println(purse+"\n");  
        
        purse.addQuarters(3);
        purse.addNickels(2);        
        System.out.println(purse+"\n");
    }
}

and make sure that you get the following output:

Quarters: 1
Dimes: 2
Nickels: 3
Pennies: 4
Value: $0.64

Quarters: 4
Dimes: 2
Nickels: 5
Pennies: 4
Value: $1.49

 

Now add a method with the following header:

public void addMoreCoins(Coins other)

This is a mutator method that is much like a combination of all of the "add" methods.  The parameter is another Coin object and the goal is to add all of the quarters from that object to the quarters instance field, all its nickels to the nickels field, etc.  Don't worry about where the parameter Coins come from.  Your job involves calling its "get" methods and using the returned value to help modify your instance fields.

 

Once you think that you've done that properly, change your runner to be the following:

public class CoinRunner
{
    public static void main()
    {
        Coins purse = new Coins(1,2,3,4);        
        System.out.println(purse+"\n");  
        
        purse.addQuarters(3);
        purse.addNickels(2);        
        System.out.println(purse+"\n");
        
        Coins change = new Coins(2,1,0,0);        
        purse.addMoreCoins(change);       
        System.out.println(purse+"\n");            
        
        Coins evenMoreChange = new Coins(0,0,2,4);     
        purse.addMoreCoins(evenMoreChange);        
        System.out.println(purse);       
    }
}

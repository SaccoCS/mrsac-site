Variables Programs

Complete the following programs that output Strings and ints.  Each program should use at least one variable.

1)  Write a program named GoTeam, which will declare and assign your favorite sports team in a String variable.  It should then concatenate and print out a String in the form "GO <teamname>!!!"

For example:
GO Chargers!!!

 

2) Write a program named FavColor which declares and assigns variables for your name and favorite color. Then you should print out your name on the first line and your favorite color on the second line below.

For example:
My name is Willie.
My favorite color is Blue.

 

3) The old board game Clue had people trying to guess three things about a murder: the murderer's name, the location of the murder, and the murder weapon.  Write a program named Clue, which will declare and assign a String variable for the name, location, and weapon you think was used.  Then print them out in the following fashion:

I have deduced that the murder was committed by Professor Plum in the Kitchen with the Candlestick

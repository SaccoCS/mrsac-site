The Stats Class

 Create a class named Stats, which will have as its instance field an ArrayList<Integer>. The purpose of the stats class will be to calculate some basic statistical measurements of the data stored in the ArrayList<Integer>.

Create a default constructor to make sure that the instance field is set up properly to have an empty ArrayList<Integer>

public void insert(int num)

The purpose of this method is to add num to the ArrayList at a location that keeps the list in ascending order. Meaning that if the List had two values, 1 and 3, a call to insert(2) would leave the List with elements 1,2,3 in order. Use a loop to determine the proper index to add the value. Once this method is working properly, you should only use this method to add elements to the List. Otherwise they could get out of order which would break your code.

 
public String toString()

Override the toString method so that it returns the toString of the ArrayList. This is an easy way of tellling what is in the List.

 

Use BlueJ to Construct a Stats object. Call the insert method a few times, passing in random values, and then call the toString to make sure they've been put in order.

Second Constructor:

public Stats(int[] nums)

This constructor will first create a new ArrayList into the instance field, and then loop through the parameter array and call the insert method, passing each value from the array in.  Do not call the add method, as it doesn't keep them in order.

Basic Stats:

public double getMean()

The getMean method's job is to return the average of the ints

 

public double getMedian()

The getMedian method's job is to return the median, which is the number in the very center of the data (assuming that the data is in ascending order). If there is an odd number of elements the answer is simply the number in the center index, but if there is an even number of elements you return the average of the two elements in the center.

Warning: Don't just assume that size()/2 will always give you the index that lies perfectly in the middle. Think of two simple examples, an ArrayList with size 3 and one with size 4 and see what calculation it takes to find the middle. Using simple examples as the foundation for your algorithms can save you a lot of heartache.

 

public int largestDifference()

This method will return the largest difference between  consecutive elements in the data


Variance and Standard Deviation:

Variance/Standard Deviation Explanation

public double getVariance()

To calculate the variance of a set of data, you must first calculate the mean. Once you have the mean, the variance is the average of the square of the distance to the mean.
Example: If the numbers are 1,2,3,4,5,6,7 then the mean is 4. The variance would be the average of (1-4)2, (2-4)2, (3-4)2, (4-4)2, (5-4)2, (6-4)2, and (7-4)2. Which is 28/7 = 4.

public double getStandardDeviation()

The standard deviation is nothing more than the square root of the variance. Please don't rewrite the code you wrote in the previous method.

Copy the Graph class below into your project. It’s purpose is to represent multiple Frequency objects together:

import java.util.ArrayList;
import sacco.objects.Frequency;
public class Graph
{
   private ArrayList<Frequency> freqs;

   public Graph()
   {
      freqs = new ArrayList<Frequency>();
   }
   
   //adds the given value to the list by incrementing an existing Frequency object in the list or creating 
   //a brand new Frequency object and inserts it into the list
   public void add(int val)
   { /* to be implemented by you */ }

   //adds an entire array of values to the list
   public void add(int[] vals)
   { /* to be implemented by you */ }

   //returns the most frequent value added
   public int mostFrequent()
   { /* to be implemented by you */ }

   public String toString()
   {
      if(freqs.size()==0)
         return "";
		
      String str = "";
         for(Frequency f: freqs)
            str+= "\n"+f;
		
      return str.substring(1);
   }
}

When a Graph object is first constructed it has no Frequency objects. When the add method is called, either an existing Frequency object is incremented or a brand new Frequency object will be inserted into the ArrayList in a way that keeps them in ascending order.


Methods

public void add(int val)

The purpose of the add method is to register a value into the list. If the list already contains a Frequency object with a matching value, increment its counter. If no object exists, add a new Frequency object to the list, making sure to keep it in increasing order. This is a challenging method that may require multiple loops. Don't forget that if the size of the list is 0 there could be problems.

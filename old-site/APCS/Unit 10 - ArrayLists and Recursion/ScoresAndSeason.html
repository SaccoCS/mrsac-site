The constructor takes in a String as a parameter to represent the team name. The constructor uses that name to call the static method listFactory, which will generate some random lists 

import sacco.objects.Score;
import java.util.ArrayList;
public class Season
{
   private ArrayList<Score> scoreList;
   private String teamName;

   public Season(String team)
   {
      scoreList = Score.listFactory(team);
      teamName = team;
   }

   public ArrayList<Score> getScores()
   {
      return scoreList;
   }

   public boolean isWin(Score s)
   {   /* To be implemented by you  */ }

   public Score winLossRecord()
   {   /* To be implemented by you  */ }

   public ArrayList<Score> awayGames()
   {   /* To be implemented by you  */ }

   public Score lastHomeGame()
   {   /* To be implemented by you  */ }

   public String toString()
   {
       String str = "";

       for(Score s: scoreList)
       {
           if(isWin(s))
               str+="W "+s+"\n";
           else
               str+="L "+s+"\n";
       }

       str+="\nWin-Loss: "+this.winLossRecord();
       str+="\nAway Games: "+this.awayGames();
       str+="\nLast Home Game: "+this.lastHomeGame();

       return str;
    }
	
    public static void main()
    {
        Season s = new Season("Scripps Ranch Falcons");
        System.out.println(s);
    }
}

 

Methods:

Implement each of the following methods. After each one runs, the instance fields should remain unchanged. The main method will create a Season object and then print out the toString. Use this to test and make sure that your code is functioning.

 

public boolean isWin(Score s)

This method is a utility method to help analyze the parameter Score object and returns true if the team defined in the instance field won the game. The tricky part is that you don't know if your team is the home team or the away team

 

public Score winLossRecord()

This method will determine the win-loss record for the Season. It will return the result as a new Score object where the away team’s name is “Wins” and the home team’s name is “Losses” and the number of wins/losses are the points. You may assume that there are no ties.

 

public ArrayList<Score> awayGames()

This method will return an ArrayList of all the Scores where our team is away.

 

public Score lastHomeGame()

This method will return the lastHomeGame of the season. You may assume that there is at least one home game.

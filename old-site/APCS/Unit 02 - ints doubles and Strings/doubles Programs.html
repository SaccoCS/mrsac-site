double Programs

1) CelciusToFarenheit

Creat a program named CelciusToFarenheit, which will have two methods, one named CtoF and one named FtoC.  use the following structure:

public class CelciusToFarenheit
{
   public static void CtoF()
   {
      
   }
   
   public static void FtoC()
   {
   
   }
}

Each method should first prompt the user to enter a temperature in one of the systems and give the equivalent temperature in the other system.  Use the two formulas below:

F = ( C x 9/5 ) + 32
C = ( F - 32 ) x 5/9

For example, If CtoF was run, and the user entered 23, the output would be:

23C is equivalent to 73.4F

And if FtoC was run and the user entered 75, the output would be:

75F is equivalent to 23.88888888888889C

 

2)  WinLossRecord

Create a program named WinLossRecord, which will prompt the user for a number of wins, and a number of losses, and return the percentage of the total games that have been won. 

For example:  If the user has 6 wins and 4 losses, the output would be:

 6 wins and 4 losses is a 60.0% win rate

You must keep the number of wins and losses as ints.  You will not get full credit if there are decimals (i.e. 6.0 instead of 6)

 

3)  CheckSplitter

Create a class named CheckSplitter, which will ask the user for three things:  The cost of the bill, what percent to tip, and how many people are splitting the bill.  It will then output the toptal amout that each person owes in order to cover the check with the tip included.

For example:  If the bill is $58, with an 18% tip split amongst 3 people the output would be:

Including a tip of 18%, a $58.0 bill would cost each of the 3 people $22.813333333333333

Don't worry about the formatting... it happens.

 

4) PastaRecipe

Mr. Sacco's pasta recipe calls for 3 cups of unbleached flour for every 4 eggs.  Create a program named PastaRecipe, whose main method will ask how many eggs are being used, and then return how many cups of flour should be used.

For Example:

How many eggs are you using?
13

You should use 9.75 cups of flour

 

5) GasRemaining:

Create a program named GasRemaining, which will prompt the user for how much gas is currently in your tank, how many miles per gallon the car gets, and how far you want to drive.  The program should print out how much gas will be left in the tank when you finish the drive.  For this program you may take in all values as doubles, since they aren't necessarily whole number values.

For Example:

How much gas is in your tank (in gallons)?
9

How many miles per gallon does your car get?
28.5

How far do you want to drive (in miles)?
200

After starting with 9 gallons, and driving for 200 miles at 28.5 mpg there will be 1.98245614 gallons remaining in the tank.  That is enough for 56.5000000000004 more miles.

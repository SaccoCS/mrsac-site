Arithmetic Shortcut Operators

When dealing with ints or doubles it is often the case that you will want to add or subtract from some variable and have the result be stored back inside the original vairable.

Consider the following:

int numOfStudents = 32;
numOfStudents = numOfStudents + 4;

The stcond line in this snippet of code changes the numOfStudents variable to be 4 larger than is currently stored in it.  There is a big difference between

numOfStudents = numOfStudents + 4;

and

numOfStudents + 4;

The second statement doesn't change any variable.  It may calculate the value of 36, but without an equals sign that value goes nowhere.

 

Shortcut!

Since increasing a value is so common, shortcut operators have been created:  They are +=, -=, *=, /=, %=, and using them will first cause an arithmetic operation to occur, and then assign the result back into the variable.

Thus:

numOfStudents = numOfStudents + 4;

can be written as:

numofStudents += 4;

This works for most arithmetic, including mod.

 

For example, Consider the following:

int num = 5;
num *= 3;
System.out.println(num);
num -= 2;
num /= 3;
System.out.println(num);

The output of this code would be:
15
4

This is the same as writing the following:

int num = 5;
num = num * 3;
System.out.println(num);
num = num - 2;
num = num / 3;
System.out.println(num);

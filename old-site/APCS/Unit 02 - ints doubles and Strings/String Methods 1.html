String Methods 1:

Complete each of the following programs by getting a String from the user and calling one of the String methods to help you get the desired result.  Remember that Strings are objects, meaning that the methods you'll call should be called using a String variable instead of directly from the String class.

1) NameMeasure - prompts the user to input their name and prints out a sentence that indicates how long the name is.

Sample input: Sacco
Output: Hello Sacco,  There are 5 letters in your name.

2) Vertical5 - takes in a 5 letter String from the user and uses 5 printlns to print a letter on each line.  You may assume that there are exactly 5 letters in the String.

Sample input: Sacco
Output:
S
a
c
c
o

3) FirstAndLast - takes in a String from the user and outputs the first and last letters of that String.

Sample input:  Mister Sacco
Output: M o

4) EndChopper - takes in a String from the user and prints out the String with the first 3 letters and the last 3 letters removed.

Sample input: Mister Sacco
Output: ter Sa

5)  HalfAndHalf - takes in a String from the user and uses two printlns to output the first half of a String on one line, and the second half on the following line.  You may assume that the number of letters in the String is even.

Sample input:  abcdefghijklmn
Output:
   abcdefg
   hijklmn

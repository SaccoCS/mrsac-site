String Helper Method Programs

Each of the following programs should have a main method, and also a helper method in the same class.  Start by creating the helper method and focusing on getting it functioning the way you want it to.  After that create a main method that uses the helper method to complete the given task.

 

1)  UpperLower:  This program will take in a five letter String from the user, and then print out each letter several times per row with alternating caps and lowercase.

Example:
Please enter a 5 letter String:
apple

AaAaAaAaA
PpPpPpPpP
PpPpPpPpP
LlLlLlLlL
EeEeEeEeE

 

To do this create a method with the following header :

public static String getUpperLower(String letter)

The goal of this method is to take the variable letter, find its upper case and lower case versions, and then put them together at least nine times in a String.  Be sure to test your program and make sure that putting a parameter of "c"  will return "CcCcCcCcC"

Once the getUpperLower functions properly, write a main method that prompts the user to input a five letter String and then uses the getUpperLower method to help produce the correct output.

2) Shifter:  This program will take in a five letter string from the user and then print out a shifted version (the last letter is moved to the front)  five times

Please enter a five letter word:
apple

eappl
leapp
pleap
pplea

 

To do this, create a helper method with the following header:

public static String shiftOneLetter(String str)

The purpose of this method is to return a new string that is made up of str, but with the last letter located in the front.
Test out this method using BlueJ, passing in "hello" should return "ohell"

 

Once the shiftOneLetter method functions, write a main method that prompts the user to enter a String, and then keeps printing out a shifted version over and over.   It is important that you save the previous shifted version so that you may use it when you call the shiftOneLetter method again.

3) PigLatin:  This program will ask the user for a five word sentence, and then print out that same sentence in pig latin.

For Example:
Please enter a five word phrase:
my pig speaks funny latin

In pig latin that is:
ymay igpay peakssay unnyfay atinlay

 

In order to accomplish this you will be creating three different helper methods to assist your main method.  One to convert a single word into pig-latin, and two to help chop the sentence into single words.

Start by creating a class named PigLatin.  Then add the following helper methods:

- Create a method named toPig, which takes in a String as a parameter and returns a String that is the Pig Latin version of the parameter. 

For those that don’t know how to speak pig latin, the first letter is moved to the back of the word, and then it attached an “ay” to the end. For example, hello becomes ellohay.  In reality this only works for words that start with a consonant and then a vowel, but the other rules require a little more knowledge of programming than we currently have so we’re going to pretend that it has simpler rules.

Test your method with several words of different lengths to make sure that it works properly with one word.

 

- Create a method named firstWord, which takes in a String as a parameter and returns the first word of that String.  For example, the String "Sacco is amazing" will return simply "Sacco".  This accomplished similarly to how you've done it before using indexOf to locate the space.  You may be afraid of what happens to the rest of the parameter, but that's not this method's problem.

You may assume that the parameter contains at least one space, meaning the indexOf method will return a good value.

Test this method out completely before moving on.

 

- Create a method named postFirst, which is the complement of the firstWord method that you just wrote.  It takes in a String as a parameter and returns a String that contains everything but the first word of the parameter String.  For example, the String "Sacco is amazing" will return "is amazing".  Make sure that you do not include the space this new String.  Start your substring on the index after the space.

You may assume that the parameter contains at least one space, meaning the indexOf method will return a good value.

Test this method out completely before moving on.

 

- Now it's time to write PigLatin's main method.  While you write this method you should keep in mind that you have methods that will help you: convert a single word to pig latin, extract the first word from a String, and get everything but the first word.

This program should prompt the user to input a 5 word String (you may assume that it is exactly 5 words).  Your job will be to use the helper methods that you've already created to create and print out a String that contains all five words in Pig Latin.

Hint: Assuming str is a String, consider  str = postFirst(str);    Since str is both being passed as the parameter and recieving the returned value, that line of code will replace str with itself, but with the first word removed.  You will repeatedly access the first word, convert it to pig latin, and then change the String to no longer have the first word.  Once you've converted all 5 words, print out the five new words with spaces in between.

BetterCash:  This program is going to ask the user how much their bill is, and then add 7.5% tax and print out how much the total is.  This class will have a helper method that will have the ability to take a double and put it into a cash format, meaning there are exactly two digits after the decimal point.  Instead of manipulating the double directly, you are going to use String methods to accomplish the task.

 

For Example:

How much was the bill?
18.84

After 7.5% tax, a bill of $18.84 is $20.25

 

Create a helper method named cashFormat.  This method will take in a double and return a String in 'cash format', meaning there is a dollar sign and exactly two digits after the decimal point.  You will want to do this using String methods instead of casting the double.

Tip:  Since you will be taking in a double, you will want to convert it to a String.  The cheap way to do this is to add an empty String to it.  For example, if the double is named value, the following line will get a String version of it:

String valueStr = value + "";

Since a String is involved in the addition Java concatenates them together, resulting in a new String that has only the double.

From there your job will be to create and return a String that has a dollar sign and exactly two digits past the decimal.

Caution!!!!!  Some doubles don't have more than one digit past the decimal point.  This could have StringIndexOutOfBounds consequences.  You should add a "0" to the bask of the String before you start manipulating it.

Test this method out with a variety of doubles, like 4.1365213 and 9.2.

 

Once the cashFormat method works, create a main method that will prompt the user to input a cost, and then output the original cost along with the cost after tax.  Both should be formatted using the cashFormat method.

Class Hierarchy Reference

public class One
{
    public void method1() 
    {
        System.out.print("One1 ");
    }
}

public class Two extends One 
{
    public void method3() 
    {
        System.out.print("Two3 ");
    }
}

public class Three extends One 
{
    public void method2() 
    {
        System.out.print("Three2 ");
        method1();
    }
}

public class Four extends Three 
{
    public void method1() 
    {
        System.out.print("Four1 ");
        super.method1();
    }

    public void method3() 
    {
        System.out.print("Four3 ");
    }
}

The following variables are defined.

One var1 = new Two();
One var2 = new Three();
One var3 = new Four();
Three var4 = new Four();
Object var5 = new Three();
Object var6 = new One();

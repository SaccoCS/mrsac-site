ints

int division

When two ints are being divided, the result is truncated. This means that it is not rounded, but the decimal portion is simply dropped. Another way to think of this is that it gives the quotient of a division expression

For Example: if you take 20/3, the result would be an even 6 because 3 goes in to 20 six times

int division trick

A common use of int division is to take an int and divide it by ten. The result will be the original number, but with the last digit removed

For Example: 12345/10 evaluates to 1234

mod

The mod operator ( % ) represents the remainder function. When you take a mod b, the result is what's left over after you remove as many b's as you can from a.

mod uses

odd/even
A very common use of mod is to determine if a value is odd or even. When a number is even, modding by two will result in a 0, odd numbers will result in a 1. 

if( num%2 == 0)
   System.out.println(num+" is EVEN!!!");
else
   System.out.println(num+" is ODD!!!");

Value Wrap Around
The result of a%b will always be less than b, which means that you may continually increase a number, but make it so that when it passes a certain point, the number is wrapped around to zero.

for( int i = 0; i <= 15; i++)
{
   int num = i%4;
   System.out.print( num );
}

Even though the loop above has i go to 15, the number printed is limited to be less than 4, the resulting numbers printed would be: 0123012301230123

Last Digit
Similar to dividing by 10, modding by ten has a use too. When a number is modded by 10, the result is the last digit in the number. For example: 12345%10 would result in a 5.

Consider the following code, which will sum up all of the digits in a number using both the /10 trick and the %10 trick

public static int digSum(int number)
{
   int sum = 0;
   while(number > 0)  //keep "removing a digit" until you hit 0
   {
      sum += number%10;  //add the last digit
      number /= 10;  //removes the last digit from number
   }
   return sum;
}

doubles

doubles are primitive values used to represent decimal numbers in Java.

Compatibility with ints

It is often the case that you must mix doubles and ints together. You be careful to respect one fact: Every int may be stored as a double, but not every double may be stored as an int.

int intVal = 77;
double dubVal = 77.5;

int a = dubVal; //ERROR: Possible Loss of Precision
int b = intVal; //perfectly fine.  All ints are automatically converted.

Casting between ints and doubles

Sometimes you would like to consider an int as a double or vice versa. This is when casting is used.

Casting a double to an int:
When casting a double to an int, the value is truncated, meaning that the decimal portion is completely dropped.

double d = 7.12345;
int num = (int)d;

After the sample code above, the value of num is 7. This is most commonly used when dealing with random numbers (See Below).


Casting an int to a double
Since every int may be represented as a double, the only reason to cast to a double is to force double-division.

int a = 25;
int b = 2;
double val1 = a/b;
double val2 = (double)a/b;

In the code above, val1 is the result of dividing two ints together, so it has a value of 12. For val2 a was cast to a double before the division, so that means that double-division was used. The result is that val2 contains a 12.5

Math Methods

You need to be aware of how to use the following methods, as they may be used in some calculations

double Math.abs(double val)

The abs method calculates the absolute value of an int or double. This is sometimes useful when trying to calculate the largest difference between two values. If you have two ints, a and b, a-b and b-a are opposite values and would require an if statement to get a positive difference. Using Math.abs(a-b) will always give you a positive value, no matter which int is larger.

double Math.pow(double base, double exp)

The pow method is useful when calculating exponents. You simply give it a base and an exponent and it will return baseexp

double Math.sqrt(double val)

sqrt is another utility method used to help find the square root of some value. For example, the following code uses a combination of Math.pow and Math.sqrt to represent the distance formula

double dist = Math.sqrt(Math.pow(x2-x1,2)+Math.pow(y2-y1,2));

double Math.random()

The random method returns an int from 0 to 1 exclusive. This is mostly used for calculating random ints

Random Numbers

int rand = (int)(Math.random()*howMany)+startVal;

Random numbers are useful for many purposes.

Deciding between choices:

int val = (int)(Math.random()*3);
if(val == 0)
  System.out.println("Choice A");
if(val == 1)
  System.out.println("Choice B");
if(val == 2)
  System.out.println("Choice C");

Be careful to recognize that the random number above has a minimum value of zero. If you wanted the numbers 1-3, there would be a +1 tagged on the to the end of the random number calculation.

Random Index

public String randomVal(String[] arr)
{
   int index = (int)(Math.random()*arr.size);
   return arr[index];
 
}

In this case, the fact that the random number starts at zero works to our advantage since that is the first index of the array. Also, since the size of the array is used as a multiplier, the random number will be less than that value, which will keep it in bounds.

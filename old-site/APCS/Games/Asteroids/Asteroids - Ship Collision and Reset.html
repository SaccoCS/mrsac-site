Ship Collision

The entire purpose of the Asteroids game is that the ship needs to avoid the Asteroids, but so far that has not been a part of our game.  Go to the AsteroidsBoard and add a method named shipVsAsteroids.  This method should return a true if the ship's hitbox overlaps any of the Asteroids' hitboxes.

Create a method named stopGame, which will pass a false into the setActive method of all the Asteroids, the Bullets and the ship.  This should cause the game to pause right where it is.  THIS WILL NOT WORK PERFECTLY.  Because of some design flaws, the Asteroids will still spin and you'll still have the ability to shoot.  This will still earn full credit.

Call shipVsAsteroids from the onTimerTick method, and if it returns a true call the stopGame method.  Test out your program to make sure that an Asteroid hitting a ship is causing the game to stop.

At this point it might be a good idea to change the game so that it starts out with a few Asteroids.  Otherwise you might find that the game is a little overbearing.

 

Reset

Add a method named reset which will set the ship back to the original position, clear all the Bullets from the list, clear the Asteroid list and add some new ones.  This method should also make sure that the ship is set back to active.

Go to the onKeyPress method and make sure that when R is pressed (keyCode 82) that the reset method is called.

 

 Leveling System

To make this more like a game, add a level system.  Add an int to represent the level, and start it at 1.  Make sure that you've used PaintBrush's drawString method to display the level on the screen (don't forget about the setFontSize and setColor methods that PaintBrush has).

During the onTimerTick method, check to see if the Asteroid list is empty.  If it is, increment the level and add more asteroids.  It works best if the number of Asteroids added is determined using the level with a formula like 2*level+2. 

You will also have to modify the reset method to set the level back to 1 and make sure that the starting number of Asteroids is appropriate for level 1 of you game.

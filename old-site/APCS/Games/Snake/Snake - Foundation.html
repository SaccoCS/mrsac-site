Snake

One of the first well known cell phone games was known as Snake. The game involves guiding a 'snake' around the screen, trying to eat the food that appears randomly. Once the snake gets the food, it grows a little longer, making its body harder and harder to avoid.

This game will be built using a SaccoWindow as the engine. Copy the following class into your project.

There are various methods that help the SaccoWindow function. The onKeyPress will help us react to the keyboard, the onTimerTick will help with the timing, and the paintWindow gives us a way to display what we want. If you run the program now, you will see a 641x641 window with nothing but a white background. You can see an online example of it here.

paintGridLines

You may notice that one of the methods is not a standard method. The paintGridLines method is a helper with the purpose of drawing lines at even intervals so that the window can be divided up into a grid. Fill in the method so that the Canvas parameter is used to draw a set of horizontal and vertical lines to break the window up into a grid. Notice the instance fields that I have already given you. They define the spacing between each row/col and how many of each there are. Use these to help your loop.

Square

One of the reasons that Snake became popular on early cell phones was because visually it is simply a collection of squares. Create a class named Square, which should have instance fields for the row, column, and color of a specific square.

The constructor should take in two ints and assign them to the row/column instance fields. Assign whichever color you would like to the color field. This will be the default color of your snake.

Add a second constructor to the Square class that sets the color, but leaves the row/col as 0.

- Add basic accessor and mutator methods for each of the three instance fields.

- Add a toString that returns a String with the row and the column. For example "(3,4)"

- Add an equals method, which will take in another Square object and return true if both the row and the column match 'this' object's row/col. Color is irrelevant.

Painting the Squares

Back in the SnakeWindow class we need to add the ability to paint a square if we need. Add a method with the following header

public void paintSquare(Canvas c, Square sq)

This method's job is to fill a rectangle of the parameter square's color at the parameter squares location. The one issue that you must deal with is the fact that the square holds a row/column and you can only paint with an x/y. When converting between the two, keep in mind that (A) rows are linked to the y direction while columns are linked to the x direction (B) The size of each row/column is in the 'squareSize' variable.

In order to test out the paintSquare method, add the following three lines to the paintWindow method

paintSquare(c,new Square(0,0));
paintSquare(c,new Square(numCols-1,numRows-1));
paintSquare(c,new Square(0,numRows-1));

Run the program. Hopefully three squares appear in the corners, leaving the bottom-left blank. If you see the upper-right square is blank, you probably have a mix up between your x/y and rows/cols.

The Snake

Now that we've created the game's basic building block, it's time to organize it. Create a class named Snake, which will be in charge of keeping track of all the pieces that make up the body of the Snake.

The Snake class should have the following instance fields

- an ArrayList<Square> to hold each piece of the snake's body.

- an int to represent the maximum size that the snake should be

Snake's constructor should take in a row and a column that are going to represent the location of the Snake's first Square. Make sure that the constructor initializes your ArrayList and then add a single Square object, with a row and column that match the parameters. Also be sure to set the size variable to a number like 10 (this can change later).

Snake should have a basic accessor for its ArrayList<Square> field

Painting

Back in the SaccoWindow, add a method with the following header:

public void paintSnake(Canvas c, Snake s)

The purpose of this method is to paint the parameter snake on the given Canvas. To accomplish this, loop through Snake's ArrayList and call the paintSquare method.

To test this out, go to the paintWindow method. Delete the three previous lines that tested the paintSquare method and replace it with the following

paintSnake(c,new Snake(5,10));

Incorporating a Snake

Now that we know a Snake can be drawn, add one as the instance field to the SnakeWindow class.

In SnakeWindow's constructor, be sure to construct a new Snake into the instance field. The row and column for this Snake should be centered, so use half of the numRows/numCols variables as the initial values.

In the paintWindow method, change the call to paintSnake so that it paints the instance field Snake instead of the one that's being generated on the spot.

Once again, run your game. This time the snake that appears should be somewhere close to the center (there's an even number of rows/columns, so there is no exact center).

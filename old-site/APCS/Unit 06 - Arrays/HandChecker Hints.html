Hints for HandChecker
Please attempt to devise your own strategy before checking.  Stretch your brain!

 

isFlush: use an early return to return a false when you find two cards with differing suits.

isPair and isThreeOfAKind:  These two methods are similar.  While looping through your array, use a nested loop to add up the number of cards that match your ‘current’ card’s value.  Then observe that counter to see if it’s 2 or 3

isTwoPair:  One way to complete this is to treat it like a single pair, bud add an extra variable: String pairVal = null; When this value is null it means that you haven't found a pair yet. When you find your first pair, assign its value to pairVal. When you come across another pair, you will be able to check its value against pairVal's to make sure that its different. 

isStraight:

Use a for loop to find the smallest rank in the array (I'll call it smallRank)

Loop from smallRank through smallRank + 4 and each time search for that rank within cards (boolean flag). If that rank is not present in those cards, return false.

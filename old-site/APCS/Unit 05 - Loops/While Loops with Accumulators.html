While Loops with Accumulators

Sometimes a loop is created for the purpose of gathering or manipulating a piece of data.  In these situations, an accumulator variable is often used.  An accumulator is a variable which is modified a little bit each time the loop comes around.  After the loop has finished, the accumulator has the desired number in it.

For example:  Write a program named Summer, which will repeatedly prompt the user to enter numbers.  The user can trigger the end of the loop by entering the word "exit".  The program will then print out the sum of the numbers that the user has entered.

First, let's get the basics of the loop going.  We need to set up a loop that asks the user for a value and stops once the value is "exit"

import java.util.Scanner;
public class Summer
{
    public static void main()
    {
        Scanner scan = new Scanner(System.in);
        System.out.println("Please enter a number, or type exit");
        String input = scan.nextLine();
        
        while(!input.equals("exit"))
        {
            
            System.out.println("Please enter a number, or type exit");
            input = scan.nextLine();
        }
    }
}

Notice that the input variable is in charge of the loop.  The first value is stored into it, and then it is replaced each time the loop comes around.  As long as the input variable isn't "exit", it should continue looping.

 

Accumulator Time!

In order to keep track of the sum of all the numbers entered, we do not need to keep track of the individual values, just a running total of all of them.  To do this, we will declare a single variable named sum, and as the loop continues we will add each value on to the sum.

 

import java.util.Scanner;
public class Summer
{
    public static void main()
    {
        Scanner scan = new Scanner(System.in);
        System.out.println("Please enter a number, or type exit");
        String input = scan.nextLine();
        
        int sum = 0;
        
        while(!input.equals("exit"))
        {
            int num = Integer.parseInt(input);
            
            sum += num;
            
            input = scan.nextLine();
        }
        
        System.out.println("The sum is: "+sum);
    }
}

Assume that the user enters the input 3,6,8, and exits.  The sum variable would start at 0 before the loop, adding the 3 would make it 3, adding the 6 would make it 9, and adding the 8 would make it 17.  Take note that the sum variable is not finished until the loop is completely over; only at that point we can print out the sum.  This is often the case with accumulators.  

 The two most often seen accumulators are variables used to sum, and variables that are used to count.

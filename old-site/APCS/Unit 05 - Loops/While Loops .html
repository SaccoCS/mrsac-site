While Loops

Complete each of the following tasks, which require a loop in some way.  In order to do this, you will have to think about a REPEATING process that will get the job done, and then design a while loop that will perform the task.


Example Problem:Write a program that will ask the user for an int, and then repeatedly calculate and print out the doubles of that number. No numbers greater than 100 should be printed.

For Example: if the number entered was 5, then 5,10,20,40,80 would be printed (the next value is 160, which is too large)

Restrictions:
You may only use one int named num to hold the number you're using.

Strategy: You should first set up the num variable. 

Scanner scan = new Scanner(System.in);
System.out.println("Please enter a number less than 100");
int num = Integer.parseInt(scan.nextLine()):  //direct parsing like a boss

Now that num is set, you can first start thinking of what code could I use over and over to print out the doubles of that number. 

System.out.println(num);
num = num*2;

This code has two parts: First it prints out the number, and then it prepares num for "next time" by doubling it. If we repeated it several times, num would be constantly getting doubled and printed over and over. 

System.out.println(num);
num = num*2;
System.out.println(num);
num = num*2;
System.out.println(num);
num = num*2;
System.out.println(num);
num = num*2;
System.out.println(num);
num = num*2;
System.out.println(num);
num = num*2;
...

Notice that the num=num*2 line's purpose is to set up the num variable so that it's ready to be printed the next time. It is important that you consider "setting up for next time" when doing these programs.

Problem: This method will print out all the doubles, but it just prints out a bunch of them without caring if the number is too large. Since one of the rules is to not print it out when the number is over 100, we'll add an if statement (no else) that will only allow the code to run when it's under 100. 

if(num<100)
{
   System.out.println(num);
   num = num*2;
}
if(num<100)
{
   System.out.println(num);
   num = num*2;
}
if(num<100)
{
   System.out.println(num);
   num = num*2;
}
if(num<100)
{
   System.out.println(num);
   num = num*2;
}
if(num<100)
{
   System.out.println(num);
   num = num*2;
}
...

Now when running this program, it will only print out the values as long as they're less than 100. Once they pass 100, the rest of the if statements will be blocked from running.

 

Looking at the code above, we can turn it into a while loop to make our lives easier:

while(num < 100)
{
   System.out.println(num);
   num = num*2;
}

This is the same as writing an infinite amount of if statements.

 

The num < 100 is very important.  You must think hard about when you want the loop to stop if you want to avoid an infinite loop. 

 

Assignments

The general strategy of each of the programs below is to start by thinking of some code that will repeatedly perfom some task with a variable, and then set the variable up for the "next time".  With loops having to many variables can be an issue, so I will be restricting the number of variables that you may use.


CountTo10:This method will prompt the user for an int that's less than 10. Then it will print out increasing numbers until it hits 10.

For Example:
Please enter a number:
6
6 7 8 9 10

Restriction:  After you get the user input you may only use one int variable to hold the number.

Strategy: First try to think of some repeating code that will increase and print out the number every time. Then surround it with an if that will only allow the increase/print to run if the number is 10 or less.


PrintVertical:This program will prompt the user for a String, and then print out each letter of the String one at a time.

For Example:
Please enter a String:
Hello
H
e
l
l
o

Restriction:You may only use two String variables (you only need one). One to hold the original String, and one to repeatedly hold the first letter for printing purpose (Hint: a String may be set to a substring of itself)

Strategy: First try to think of a bit of code that will print out the first letter of the string and then shorten it so that the first letter is missing, thus giving it a new first letter to print out the next time around.

Then think of an while loop that will use the String's length to only allow it to run if there are more letters to print


SuperSafeDivider – This program will ask the user for a numerator and a denominator, but this time if the denominator is zero, a loop should activate that asks the user to input a non-zero number until they actually do it.

Example:
Please enter a numerator:
9
Please enter a denominator:
0
Please enter a non-zero denominator:
0
Please enter a non-zero denominator:
4
9 / 4 = 2 remainder 1
Hint: You will ask the user for a numerator and a denominator once before the loop starts, and then use the denominator's value to keep asking until it's not zero

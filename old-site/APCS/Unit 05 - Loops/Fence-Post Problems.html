Fence-Post Problems

 

The problems below are fencepost problems, meaning that we need to do something between each element, but not after the last one or before the first one.  There are two strategies that people use:

-  Stop the loop one short of the end, and then print out the last value alone

-  Go the full loop, but use an if statement to separate the last element from the rest.

 

 

1)  Create a method named listNums, which will take in an int as a parameter, and print out the numbers from 1 to that number with hyphens in between:

For Example:  if the number was 6

1-2-3-4-5-6

 

 

 2) Create a method named letterArray, which will take in a String and print out every letter of the String between curly braces {}.  All of the letters should be separated by commas.

For Example:  if the String was "Sacco"

{S,a,c,c,o}

What's a double?

A double is a primitive data type that's meant to hold decimal values.  Any time that you write a number with a decimal point Java will consider it a double.

double variables may be declared and assigned, just like ints:

double gpa = 3.93;

 

doubles can store ints:

Since every single integer value can be turned into a double easily, Java will automatically turn any int into a double if the situation calls for it.

double num = 42;

Even though the right side has an int (42), the compiler will be fine with it.  When the time comes to assign the value to the variable, the 42 will quickly be turned into a 42.0

 

 

ints can't store doubles:

The rule is that every int is technically a double (42 is 42.0, -7 is -7.0, etc), but the opposite is not true (there is not int that can represent 43.7).  This means that storing doubles as ints is impossible:

int num = 3.7;

The compiler will not allow this to happen.  It knows that double's can't be stored as ints an it will give an error that says "possible loss of precision".  The compiler doesn't even check the value!  Even if it is a number that can easily be represented as an int.

int num = 3.0;

This still won't compile, simply based on matching data types.

 

 

A double is known as a floating point number, which essentially means that it's stored in memory using scientific notation.  In memory, a certain number of bits are there to represent the sign of the number, the exponent, and the actualy numeric value.

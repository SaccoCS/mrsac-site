What is an int?

An int is one of the basic data types that we will be using in our programs.  It is short for integer, and may only be used to store whole numbers.  Any time a number is typed with no decimal and no quotes, it will be considered an int.

For example, the following snippet of code declares an int variable named num, and assigns a value of 4 to it:

int num = 4;

You may not assign any value to an int variable that is not a whole number.  For example, the following lines will not compile:

int num = 4.0;

This is a fine attempt, but even though 4.0 is a whole number including the decimal point technically changes the type from an int to a different data type that is meant to handle decimal values (a double). 

int num = "4";

In this case the value being assigned to the num variable looks like an int, but by surrounding it with quotation marks it is being transformed into a String that has the character 4 in it.  This strips the 4 of any mathematical value.  It is simple another character, such as an "A" or a "#".

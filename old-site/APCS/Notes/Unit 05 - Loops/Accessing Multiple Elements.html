Accessing Multiple Elements

 

The standard traversing for loop only accesses one character at a time.   

 for(int i = 0; i<str.length(); i++)
{
   String letter = str.substring(i, i+1);
  
}

The int i is set up to represent every index in the String.  The substring is from i to i+1, which is a single character.

 

Sometimes you need to access more than one element of a String in order to complete a task.  This is fine, you just need to do the following:

1)  Make the substring grab more letters:  If you're trying to access two letters at a time you may modify the substring to go from i to i+2, three letters would be from i to i+3, etc.

2) Shorten the Loop:  If your loop involves accessing two letters at a time, the boundary of the loop need to be modified so that you don't go out of bounds.  The problem occurs when i is the last index in the String.  The substring will try and access the character after i, but crash because it's out of bounds.  The solution is to simply shorten the loop.

for( int i =0; i<str.length() - 1; i++);

Subtracting one from the upper bound will mean that it stops one iteration sooner, avoiding the crash.

 

Example:

Write a method that will count the number of times "AP" appears in the parameter String:

public int apCount(String str)
{
   int count = 0;

   for(int i =0; i<str.length()-1; i++)
   {
      String sub = str.substring(i,i+2);

      if(sub.equals("AP"))
        count++;
   }
   return count;
}

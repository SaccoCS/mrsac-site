Strings and Escape Characters

 

What's a String?

In your first program, the text "Hello World!!!"  was printed out.  You may notice that this portion of the code was turned green by Java.  When you put text between quotes, it tells Java that you want it to be considered text, not code.  In programming, this is known as a String.  Change the code so that your program will print out "Hello Mr. Sacco".  This is done simply by changing the String to whatever you want it to say.

Most of the time you won't run into an issue changing a String, but sometimes you'll run into a problem.  For example, if you wanted to print out Hello Ian "King of the Nerds" Sacco, you would have to change the code to say:

System.out.println("Hello Ian "King of the Nerds" Sacco");

This will not compile.  The problem is that the quotation mark already tells Java that the String is done.  Java might interpret this line of code as having two Strings, "Hello Ian " and " Sacco", with King of the Nerds as code in between.  This is a problem.

How to tell Java to skip a quotation mark:

There are a few special cases that you can put inside of a String that will make it act differently.  If you want to "skip" a quotation mark, you only need to put a backslash in front of it.  Change the HelloWorld code to say:

System.out.println("Hello Ian \"King of the Nerds\" Sacco");

Run the program.  Notice that the slashes don't appear.  This is because \" is an Escape Character that means "This quotation mark is not code, only a part of the String"

 

Other Useful Escape Characters:

 

\n
The hard return: Placing this in a String causes the rest of the string to start on a
new line, as if Enter had been pressed

Example: System.out.println(“Hello\nWorld”);

Output:
Hello
World

\t
Tab: Placing this into a String causes the text to be spaced out as if you had hit
the Tab key.

Example: System.out.println(“Hello\tWorld”);

Output:
Hello        World

\\
Backslash: If you want to place a backslash into a String without Java thinking
that it is the beginning of another escape character, you must do it twice.

Example: System.out.println(“\\SLASHES\\");

Output: \SLASHES\

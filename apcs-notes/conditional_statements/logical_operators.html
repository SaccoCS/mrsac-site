The Logical Operators

The Logical Operators are symbols that represent operations that look at boolean values and generate separate booleans. They are useful for combining boolean statements together and making more complex decisions.

The AND Operator &&

The AND operator looks at the boolean values to the left and the right and evaluates to true if the value on both sides of it are true. There are only four possible situations.

true && true   --> true
true && false  --> false
false && true  --> false
false && false --> false

The only way to get a true value from an AND statement is when both values are true.

A classic example of this is determining if a number is between two values. Let's say that we are asking a user to input a number between 1 and 100, but the user sometimes doesn't read directions and might put in values that are way outside of that range. In order to make sure that the program runs properly, we may need to validate that the input is actually between 1 and 100.

In order to check if the number is between 1 and 100, two things must be true at the same time. The number must be both greater than or equal to one AND less than or equal to 100.

int num = //some value from user
if( num >= 1 && num <= 100)
{
   System.out.println(num + " is IN RANGE");
}
else
{
   System.out.println(num + " is OUT OF RANGE");
}

The value of num that the user enters will fall into the Goldilocks categories: too small, too large, or just right. Let's examine what happens in all three situations, substitute in some sample values for num

Too low: -22

If we substitute -22 into our boolean expression, we get:
-22 >= 1 && -22 <= 100
-22 is not greater than 1, but it is less than 100. This means we will get one false and one true.
false && true
The AND statement will looks at the two values and evaluate to false, and cause the println to be
-22 is OUT OF RANGE

Too high: 128

If we substitute 128 into our boolean expression, we get:
128 >= 1 && 128 <= 100
128 is greater than 1, but it is not less than 100. This means we will get one false and one true.
true && false
The AND statement will looks at the two values and evaluate to false, and cause the println to be

Just right: 85

The OR Operator ||

The OR operator looks at the boolean values to the left and the right and evaluates to true if the value on either side of it are true. There are only four possible situations.

true || true   --> true
true || false  --> true
false || true  --> true
false || false --> false

The only way to get a false value from an OR statement is when both values are false.

One example of using the OR operator is a pass-fail course that has two ways of passing. The professor has decided that if a student gets an A on the final they deserve to pass no matter what their grade is in the rest of the course. Otherwise they will pass if they get 60% or higher in the class. The basic structure of the if statement could look like the following

if( classPercent >= 60 || finalPercent >= 90 )
{
   System.out.println("Passing Grade");
}
else
{
   System.out.println("Failing Grade");
}

Again there are several situations that we must consider. The result hinges on the boolean expression:

classPercent >= 60 || finalPercent >= 90

The way that an OR statement works, we only need one of the expressions to be true in order to result in a true. This means that we only need either classPercent >= 60 to be true or finalPercent >= 90. This is not as strict as the && which would require both to be true.

This means that the student passes if they have:
- A high class percent but a low test percent
- A low class percent but a high test percent
- A high class percent and a high test percent

The only way that the student would fail is if they have:
- A low class percent AND a low test percent.
This is the only situation that would make both sides of the OR operator false, and thus evaluate to false.

The Negation Operator !

The exclamation point represents the Negation operator, which performs an action similar to the 'minus-sign' in normal algebra. You put the ! in front of a value to switch it to its opposite. In boolean algebra, this means switching from true to false or false to true. Consider the simplified example:

boolean b1 = true;
boolean b2 = !b1;
boolean b3 = !(b1 && b2);

The way this is written, the b2 variable should have the opposite of the b1 variable. Since b1 is true, its opposite is false, which is assigned to b2

The last calculation first asks us to evaluate the parentheses
b1 && b2 --> true && false --> false
The result is the opposite of the result
!false --> true

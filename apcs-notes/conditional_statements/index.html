Conditional Statements

This unit is all about using boolean statements to make decisions within our programs. There will be a focus on the differences between the primitive variables like ints and doubls, and Object variables like Strings and other Objects.

Unit Goals

  • Understand the boolean primitive type
  • Learn how to generate booleans using the Relational Operatos and Logical Operators
  • Utilize if statements to make complex descisions in our programs
  • Create Objects with boolean accessor methods and use them within an if statement
  • Analyze complex boolean expressions

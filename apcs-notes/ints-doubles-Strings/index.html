ints, doubles, and Strings

This unit deals with the three main types of data that we are going to deal with for the entire year. We will look at what each type can do and explore how they relate to each other. Along with that we will explore how to use helper methods, as well as write a few of our own.

Unit Goals

  • Understand the difference between ints and doubles, and how casting affects each.
  • Work with int, double, and String variables.
  • Understand the link between parameters and return types in a method header and how they are used within a program.
  • Work with the static methods of the Math class to help perform calculations.
  • Work with the non-static methods of the String class to help perform String manipulation.

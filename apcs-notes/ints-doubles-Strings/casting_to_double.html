Casting to a double

Casting is an operation that allows us to temporarily consider a certain piece of data to be a different type. This means that we can take an int and have it considered to be a double during calculations. This is most useful when you want to overcome int division

int a = 5;
int b = 2
double div = a/b; // int division -> 2

In the example above, the programmer wanted to divide a and b to get a 2.5, but since the variables are both ints, the result of the division will be truncated and give a 2.0. Sometimes truncation is fine, but often times you want to overcome the fact that the numbers are ints and do proper division.

Casting

Since the problem above is that both of the variables are ints, one solution would be to cast either of the two variables so that it is temporarily considered a double, then the division would not be truncated.

int a = 5;
int b = 2
double div = (double)a/b; // double division -> 2.5

Casting affects the value immediately to its right. In the example above, the value of a is considered a double, which prevents the division from truncating and calculates the expected 2.5 instead.

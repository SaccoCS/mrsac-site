Basic Math Operators

Java uses the + - * / % symbols for addition, subtraction, multiplication, division, and mod. The two new topics you should be aware of are Mod and Division with ints.

The Assignment Operator =

The 'equals sign' is known as the Assignment Operatorused in Java whenever you are assigning a value to a variable.

int x = 5;

The assignment operator is a right-to-left operation. The left side should always be the variable that you want to assign the value to. The example above is how we initialize a variable to its very first value. It can also be used to change the value of a variable that already has a value.

int x = 5;
System.out.println("Current Value:" + x);
x = 22;
System.out.println("Current Value:" + x);

Output:
Current Value: 5
Current Value: 22

Even though the println statement prints the value of x, in between the printlns the value of x is changed to a different value

Performing Math Operations

When using symbols to perform math we always want to make sure the result goes somewhere. That's why we frequently use an assignment statement at the same time

int a = 5;
int b = 7;
int sum = a + b;

The last line does two things. First it calculates a + b to get a value of 12. Once that value is done calculating, the result is assigned to the sum variable. Remember that with these statements, the right side is calculated first.

Making Changes to a Variable

When you perform a calculation, you don't always want to create a brand new variable. Sometimes you simply want to increase a variable.

int num = 5;
int bigNum = 100 * num;

In the last line here, int bigNum = 100 * num; the value of num doesn't change. The only way to change the value of num would be to have it to the left of the assignment operator. Which is completely legal. Consider what the following code does.

int num = 5;
num = 100 * num;

The line num = 100 * num; is confusing at first, but just remember that it's actually two things:
First the right side is calculated (500), and then the result is assigned back into the num variable. The affect is that the value of num has been changed from 5 to 500.

Combination Assignment Operators: += -= *= /= %=

Each of these assignment operator performs two functions. First it does some math, and then it assigns to the variable on the left. Consider the code below, which assigns an initial value of 20 to the num variable, and then uses the combination assignment operator to increase it by five.

int num = 20;
num += 5;

The num += 5; statement is a shortcut for num = num + 5;. The result is that the value of num is increased from 20 to 25.

Consider the following code that uses many of the combination assignment operators to repeatedly change the value of the x variable

int x = 5;
System.out.println("Current Value: " + x);
x += 4;
System.out.println("Current Value: " + x);
x *= 6;
System.out.println("Current Value: " + x);
x /= 3;
System.out.println("Current Value: " + x);
x %= 5;
System.out.println("Current Value: " + x);

Output:
Current Value: 5
Current Value: 9
Current Value: 54
Current Value: 18
Current Value: 3

Increment and Decrement Operators ++  --

Another very common procedure with an int variable is increasing or decreasing it by exactly 1. This is known as incrementing and decrementing.

Assuming we have a variable num, we could increment it using either of the strategies below.

num = num + 1;
num += 1;

But there is another shortcut that we can use that performs the same process with fewer characters.

num++;

The ++ is known as the increment operator and its sole purpose is to increase the value of the variable to its left by exactly one.

Similarly, num-- is using the decrement operator and it will cause the value of num to decrease by exactly one.

Example

Below you will find code that repeatedly makes a change to x and then prints out the new value of the variable. Towards the end it gets a little tricky. Make sure that you understand how each operation affects the x variable.

int x = 10;
System.out.println("Current Value: " + x);
x += 4;
System.out.println("Current Value: " + x);
x /= 2;
System.out.println("Current Value: " + x);
x -= 4;
System.out.println("Current Value: " + x);
x++;
System.out.println("Current Value: " + x);
x = x + x;
System.out.println("Current Value: " + x);
x--;
System.out.println("Current Value: " + x);
x *= x - 2;
System.out.println("Current Value: " + x);

Output:
Current Value: 10
Current Value: 14
Current Value: 7
Current Value: 3
Current Value: 4
Current Value: 8
Current Value: 7
Current Value: 35

Example

What are the final values of a and b after the following code runs?

int a = 5;
int b = 2;
a += b;
b -= a;
a *= 2;
b += a-4;

I would not recommend trying to do this in your head. It is possible, but there are lots of places that you could easily lose track of. Instead I recommend keeping track of the current values of a and b on a piece of paper. Every time the variable changes, cross it out on that piece of paper and replace it with the new value. That way you will always know the up to date value of each variable.

In the end a and be have a value of 14 and 5 respectively.

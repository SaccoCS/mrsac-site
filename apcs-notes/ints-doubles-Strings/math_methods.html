Math methods

The Math class is a library of methods that are useful for performing math operations. Each method is static, meaning that you use the class name when calling the methods i.e. Math.sqrt(num)

Absolute Value: abs

public static double abs(double val)
public static int abs(int val)

The abs method is used to compute the absolute value of a number. There are two method headers above because there is actually an abs method designed to take in each type of data. This means that if you give the method an int, it will return an int, giving a double will return a double.

Math.abs(3) --> 3
Math.abs(-5) --> 5
Math.abs(-9.9) --> 9.9

Example

In a normal curve, about 68% of the data falls within one standard deviation of the mean. If the mean is 10 and the standard deviation is 2, then the values within one standard deviation include 8,9,10,11,12. If you're given the mean along with a sample value, you can tell how far apart they are using absolute values.

|mean - value| = distance

The absolute value is to make sure that you don't get a negative distance. Once you have the distance, you can tell if it's less than the standard deviation.

double mean = 10;
double value = //some value from the user.
double distance = Math.abs(mean - value);

From here you could use if statements to determine if the value is within a standard deviation.

Square Root: sqrt

public static double sqrt(double val)

The sqrt function is used to calculate the square root of a value. It takes in a double and it returns its square root.

Math.sqrt(2) --> 1.4142135623730951
Math.sqrt(4) --> 2.0
Math.sqrt(9.5) --> 3.082207001484488
Math.sqrt(16) --> 4.0

This method says that it only takes in a double, but don't forget that all ints are accepted as doubles.

Example

One special right triangle is the 45-45-90 triangle. If one of the legs of a 45-45-90 has a value o x, you can calculate the hypotenuse of the triangle with x√2.

double leg = //some value from the user.
double hyp = leg*Math.sqrt(2);
System.out.println("The hypotenuse is " + hyp);

Calculating Powers: pow

public static double pow(double base, double exp)

The power function is for calculating exponents. There are two parameters that you must provide, a base and an exponent which will be calculated as baseexponent

Math.pow(3,2) --> 32 --> 9.0

Math.pow(2,10) --> 210 --> 1028.0

Math.pow(1,99) --> 199 --> 1.0

Math.pow(4,0.5) --> 40.5 --> 2.0

This last calculation shows that in a pinch you can use this function as a substitute for a sqrt if you know about fractional exponents.

pow only returns doubles

The way that the pow method is defined, it will always return a double. Mathematically, if you only give it whole number values you can expect back a whole number value. This will happen when using the pow method, but the returned whole number values will still be doubles. They'll just end in .0.

int base = 2;
int exp = 3;
int power = Math.pow(base,exp); // <-- Will not compile. pow returns a double

The code above expects a value like 8, but since we always get a double we actually get an 8.0. If you're writing a program that needs exponents and you know that the result is an int, you could cast it to keep your variables neat.

int base = 2;
int exp = 3;
int power = (int)Math.pow(base,exp);

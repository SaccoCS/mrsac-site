Mouse Event Handling:

The three mouse events that you can handle:

When a mouse button is PRESSED

When a mouse button is RELEASED

When the mouse is MOVED across the screen.

 

The MouseEvent Object:

Each mouse event handling method takes in a MouseEvent as a parameter.  This is an encapsulation of the x, y, and button that was pressed (it's an int 1, 2, 3),  You can extract the information from the object using the getX, getY, and getButton (getButton will not work for when the mouse is moved).

 

 

The Handler Methods:

You only need to override the ones that you are going to use.

@Override
public void onMousePress(MouseEvent m)

This method is called when the mouse button is pressed

 

@Override
public void onMouseRelease(MouseEvent m)

This method is called whent the mouse button is released

 

@Override
public void onMouseMove(MouseEvent m)

This method is called when the mouse moves while over the window
